## 核心条款

### 目的和依据

> 为深入贯彻落实习近平新时代中国特色社会主义思想和党的二十大精神，贯彻新发展理念，促进高质量发展，结合公司改革发展的新思路、新目标、新举措，贯彻落实中建一局全面开展精细化管理的精神，实现降本增效的目的。根据集团和公司有关规章制度，制定本办法。

### 适用范围

> 此办法适用于公司所属的国内承包类项目，不含海外项目。

### 管理原则

> 坚持项目目标责任制"责、权、利"高度统一的原则。

**逐条解释：**
- **责**：项目责任担当体成员须立足岗位职责，对项目管控目标承担明确责任，以预留保证金形式履行承诺；
- **权**：项目部在保证金预留与返还流程中享有相应权利，可按程序提出返还申请；
- **利**：项目完成目标责任后，保证金全额无息返还，并享受考核兑现奖励；未完成目标则按制度扣除。

### 职责分工

| 角色 | 职责 |
|------|------|
| **项目部** | 预留项目保证金；提交项目保证金返还申请；项目经理组织责任担当体对离任人员责任进行界定 |
| **分公司（事业部）总经理** | 审批项目保证金预留额度及预留时间（超过350万元时一事一议须经总经理办公会审议） |
| **分公司（事业部）总经济师** | 负责项目保证金的返还审批 |
| **分公司（事业部）商务系统** | 核定项目保证金预留额度、预留时间；审核项目部报送的返还申请 |
| **公司财务管理部** | 核实项目保证金预留并出具《项目保证金预留回执单》；设立台账专人归档管理；负责项目保证金返还审批及扣除执行 |
| **公司商务管理中心** | 备案《项目保证金预留回执单》原件及预留收据复印件；负责本办法解释和修订 |
| **公司资金部** | 核实工程款回收情况（竣工考核阶段）；确认项目工程款100%回款（终验阶段） |
| **公司人力资源部** | 负责员工调动事项及调动时间的确认；协助相关部门对员工调动所涉及的项目保证金预留、返还进行审核 |

## 四维宣贯表

### 做什么（8-15条）

1. **签订项目目标管理责任书**：项目开工前，项目部与公司签订《项目目标管理责任书》，明确各岗位人员保证金预留金额，作为附件具有合同效力。
2. **核算保证金预留总额**：以自施合同额为基数，采用超额累进方式计算项目保证金预留总额（基准：≤1亿部分按2.5‰、1~5亿部分按2‰、＞5亿部分按1‰）。
3. **分解落实到责任担当体**：将保证金预留总额分解至项目责任担当体成员，合计预留比例不低于应预留总额的70%；项目经理预留比例不低于25%但不高于35%。
4. **执行保证金预留缴纳**：项目部按《项目保证金预留通知》要求，在规定时限内以现金或分期方式完成保证金预留。
5. **出具并归档预留凭证**：财务管理部向预留人员出具收据，项目部将盖章的《项目保证金预留回执单》原件及收据复印件交商务管理中心备案，财务管理部设台账专人管理。
6. **竣工锁定后申请首批返还**：项目完全成本锁定、预计能完成目标收益、业主结算书已报出、资金为正现金流，且责任担当体签订《收益率达标承诺书》后，可申请返还50%。
7. **竣工考核后申请第二批返还**：项目竣工考核完成、成本复盘资料上交、资料归档完成、资金部核实工程款回收后，可申请返还40%。
8. **终验后申请全额返还**：竣工审计完成且资金部确认工程款100%回款后，返还剩余10%（全额无息）。
9. **人员调动时的保证金处理**：离任人员任职期间责任由项目经理组织责任担当体界定，保留额度参照其任职期间产值与自施合同额比例，继任者承担返还额度预留，差额由项目责任担当体共同预留。
10. **办理保证金扣除**：需扣除保证金时，由提出单位填报《项目保证金扣除确认书》和《项目保证金扣除申请表》，交财务管理部执行扣除。
11. **监督与检查配合**：接受公司对保证金预留及返还情况的跟踪、检查、抽查。
12. **离职人员保证金处理**：离职人员不再参与任何项目任何考核兑现，保证金按制度执行。
13. **中途擅离职守处理**：中途擅离职守或脱岗人员，不返还预留保证金，不享受任何考核兑现奖励。

### 怎么做（详细操作流程，10-20步）

**阶段一：保证金预留**

1. **项目部**：在《项目目标管理责任书》签订后1个月内（如为暂缓开工项目在实际进场施工后1个月内进场，以业主/监理开工令或财务账面首笔成本时间为准），向分公司（事业部）商务系统提交项目基本信息，包含自施合同额、责任担当体成员名单及岗位。
2. **分公司（事业部）商务系统**：根据自施合同额，按超额累进方式核定项目保证金预留总额——≤1亿部分×2.5‰、1~5亿部分×2‰、＞5亿部分×1‰；如总额超过350万元，启动一事一议程序报总经理办公会审议确定最高额度。
3. **分公司（事业部）商务系统**：根据责任担当体各成员承担责任和角色，分解预留比例——合计不低于应预留总额的70%，项目经理不低于25%不高于35%，确定各成员应预留金额，起草《项目保证金预留通知》。
4. **分公司（事业部）总经理**：审批《项目保证金预留通知》，明确最迟预留时间，签发后通过公司办公平台发布。
5. **项目部**：按《通知》要求组织责任担当体成员以现金方式缴纳保证金——总额≤100万元须一次性全额预留；总额＞100万元超出部分可申请分期预留，分期预留时间不得晚于工程完工百分比50%。
6. **公司财务管理部**：收到保证金款项后，核实金额，向预留人员出具收据，加盖财务专用章，出具《项目保证金预留回执单》（由财务管理部经理签字盖章）。
7. **项目部**：将盖章的《项目保证金预留回执单》原件及预留收据复印件一并递交至商务管理中心备案。
8. **公司财务管理部**：将预留信息记入保证金台账，专人归档管理，《项目保证金预留回执单》实际签字盖章时间即为实际预留时间。

**阶段二：竣工锁定后首批返还（50%）**

9. **项目部**：确认项目完全成本锁定、预计能完成目标收益、业主结算书已报出、项目资金为正现金流后，组织责任担当体签订《收益率达标承诺书》。
10. **项目部**：填写《项目保证金返还申请表》（附申请理由、项目情况说明），项目经理签字确认。
11. **分公司（事业部）商务法务部**：审核返还申请，签署意见。
12. **分公司（事业部）总经济师**：审核返还申请，签署意见。
13. **公司资金部**：核实工程款回收情况，确认资金状态，签署意见。
14. **公司商务管理中心主任**：审核申请，签署意见。
15. **公司财务管理部**：办理返还手续，将50%保证金返还至预留人员账户，所有资料留存归档。

**阶段三：竣工考核后第二批返还（40%）**

16. **项目部**：确认项目竣工考核完成、成本复盘资料已上交、资料已归档、责任担当体完成目标责任书约定的责任目标后，填写《项目保证金返还申请表》。
17. 重复流程11~15（分公司商务法务部→总经济师→资金部→商务管理中心主任→财务管理部），返还剩余40%。

**阶段四：终验后终额返还（10%）**

18. **公司资金部**：确认项目工程款100%回款后，出具书面确认意见。
19. **项目部**：凭资金部确认意见，填写《项目保证金返还申请表》，走完全部审批流程（总经济师→资金部→商务管理中心主任→财务管理部）。
20. **公司财务管理部**：全额无息返还剩余10%，结清台账。

**阶段五：人员调动与扣除**

21. **项目经理**：如责任担当体成员调动，组织责任担当体对离任人员任职期间责任进行界定，确定其保证金保留额度（保留额度=该成员任职期间产值/自施合同额×应预留金额），继任责任担当体承担返还额度预留，差额由项目责任担当体共同预留。
22. **人力资源部**：确认员工调动事项及调动时间，审核调动所涉及的保证金预留、返还。
23. **公司财务管理部**：根据《项目保证金扣除确认书》和《项目保证金扣除申请表》，执行保证金扣除，扣款记入台账。

### 怎么考核

| 考核维度 | 内容 |
|----------|------|
| **考核主体** | 分公司（事业部）商务系统、公司财务管理部、公司商务管理中心、公司资金部 |
| **考核方式** | 流程审批合规性审核（商务系统审核→总经济师/总经理审批→资金部核实→财务管理部执行）；项目目标完成情况考核（竣工锁定→竣工考核→终验） |
| **考核周期** | 随项目关键节点：①保证金预留审核（签订责任书后1个月内）；②首批返还审核（竣工锁定后）；③第二批返还审核（竣工考核完成后）；④终额返还审核（竣工审计+100%回款后） |
| **违规惩罚（量化）** | ①未按规定预留→调岗处理，对项目班子成员不予任命或免职；②超最迟预留时间1个月仍未足额预留→只发放岗薪，不发放绩效薪，直至足额预留，期间扣发绩效薪不再补发；③未按期足额预留保证金→不享受考核兑现；④中途擅离职守或脱岗→不返还保证金，不享受任何考核兑现奖励；⑤离职人员→不再参与任何项目任何考核兑现 |
| **返还条件（量化）** | 首批50%：完全成本锁定+预计完成目标收益+业主结算书已报出+正现金流+签订《收益率达标承诺书》；第二批40%：竣工考核完成+成本复盘资料上交+资料归档+工程款回收核实；第三批10%：竣工审计完成+工程款100%回款 |
| **计算公式** | 保证金总额=≤1亿部分×2.5‰+（1~5亿部分）×2‰+＞5亿部分×1‰；项目经理预留比例：应预留总额×25%~35%；责任担当体合计预留：应预留总额×≥70% |

## 核实要点清单

### 资料清单

1. **《项目目标管理责任书》**（含附件，明确各岗位保证金预留金额）
2. **《项目保证金预留通知》**（分公司/事业部总经理签发，办公平台发布记录）
3. **《项目保证金预留回执单》**（财务管理部经理签字盖章，原件在商务管理中心备案）
4. **保证金收款收据**（财务管理部出具，加盖财务专用章）
5. **项目保证金台账**（财务管理部设立，专人归档管理）
6. **《收益率达标承诺书》**（责任担当体全员签字）
7. **《项目保证金返还申请表》**（含全套审批签字记录：分公司总经济师→资金部→商务管理中心主任→财务管理部）
8. **《员工调动项目保证金返还/预留表》**（调动发生时）
9. **《项目保证金扣除确认书》及《项目保证金扣除申请表》**（扣除发生时）
10. **项目完全成本锁定资料**（财务部门出具）
11. **业主结算书报送凭证**
12. **项目竣工考核资料及成本复盘资料**
13. **工程款回收情况核实记录**（资金部出具）
14. **竣工审计报告**
15. **项目自施合同额确认依据**（不含甲供材、甲供设备、甲指分包、计日工）

### 常见问题/风险点（5-8条）

1. **保证金预留不及时或不足额**
   - 表现：《项目保证金预留回执单》时间晚于责任书签订后1个月，或实际预留金额低于应预留金额。
   - 识别方法：比对《项目目标管理责任书》签订日期与《项目保证金预留回执单》日期；核查回执金额与按超额累进公式计算金额是否一致。

2. **项目经理预留比例不符合规定**
   - 表现：项目经理预留比例低于应预留总额的25%或高于35%。
   - 识别方法：查阅《项目目标管理责任书》附件中项目经理应预留金额，计算其占应预留总额的比例。

3. **责任担当体合计预留比例不足70%**
   - 表现：责任担当体全体成员预留金额合计未达到应预留总额的70%。
   - 识别方法：汇总责任担当体所有成员预留金额，与按超额累进计算的应预留总额×70%进行比对。

4. **分期预留超期或额度不符**
   - 表现：总额超过100万元的项目，超出部分分期预留时间晚于工程完工百分比50%。
   - 识别方法：核查《项目保证金预留通知》中的分期计划与项目实际完工百分比节点。

5. **返还审批流程不完整**
   - 表现：《项目保证金返还申请表》缺少分公司总经济师、资金部、商务管理中心主任等关键节点签字。
   - 识别方法：逐页核查返还申请表上所有签字栏是否齐全，签字顺序是否符合规定。

6. **人员调动后保证金责任界定不清**
   - 表现：调出人员已离岗但未对其任职期间责任进行界定，或继任者未承担返还额度预留责任，差额未由项目责任担当体共同预留。
   - 识别方法：查阅《员工调动项目保证金返还/预留表》，核查责任界定计算过程和继任分担安排。

7. **超期未足额预留人员仍发放绩效薪**
   - 表现：项目保证金超过最迟预留时间1个月仍未足额预留，但相应人员当月仍正常发放绩效薪。
   - 识别方法：比对《项目保证金预留通知》中的最迟预留时间与财务薪资发放记录，核查超期后绩效薪是否停发。

8. **工程款回款未达100%即返还10%终额**
   - 表现：项目工程款回款比例未达100%，但已全额返还最后10%保证金。
   - 识别方法：核查资金部出具的工程款回款核实确认文件中回款比例数据，与返还金额进行对应。

### 关键指标

| 指标名称 | 计算公式 | 达标标准 |
|----------|----------|----------|
| **项目保证金预留总额** | ≤1亿部分×2.5‰+（1~5亿部分）×2‰+＞5亿部分×1‰ | 与通知核定金额一致，误差为0 |
| **项目经理预留比例** | 项目经理应预留金额÷应预留总额×100% | 25%≤比例≤35% |
| **责任担当体合计预留比例** | 责任担当体成员预留总额÷应预留总额×100% | ≥70% |
| **最迟预留时间** | 《项目保证金预留通知》载明的最迟预留时间 | 责任书签订后1个月内或实际进场施工后1个月内 |
| **分期预留截止节点** | 超出100万元部分分期预留截止时间 | 不晚于工程完工百分比50%时点 |
| **首批返还比例** | 竣工锁定后返还金额÷保证金预留总额×100% | ≤50% |
| **第二批返还比例** | 竣工考核后返还金额÷保证金预留总额×100% | ≤40% |
| **终额返还比例** | 终验后返还金额÷保证金预留总额×100% | ≤10%（须工程款100%回款） |
| **返还总额上限** | 50%+40%+10% | =100%，全额无息 |
| **超350万元一事一议** | 超过350万元的项目是否经总经理办公会审议确定最高额度 | 须有总经理办公会记录或审议纪要 |
| **人员调动保留额度** | 离任人员保留额度=（任职期间产值÷自施合同额）×应预留金额 | 有书面计算依据，项目经理签字确认 |
