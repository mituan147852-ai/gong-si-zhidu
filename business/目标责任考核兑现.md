## 核心条款

### 目的和依据

为深入贯彻落实习近平新时代中国特色社会主义思想和党的二十大精神，贯彻新发展理念，促进高质量发展，结合公司改革发展的新思路、新目标、新举措，贯彻落实项目价值创造提升行动，全面开展精细化管理，实现降本增效，特制订本办法。

### 适用范围

适用于公司所属国内承包类项目，不含海外项目。

### 管理原则

- **目标引领、价值创造**：以价值创造为导向，牵引项目实现成本最优、效益最大
- **过程激励、精准考核**：过程中及时激励，精准量化考核
- **快速兑现**：达到目标后快速兑现奖励
- **各方共赢**：达到各方共赢、成本最优、效益最大、企业持续健康发展

### 核心术语

| 术语 | 定义 |
|------|------|
| 核心责任担当体 | 项目负责人（项目经理/执行经理）、专职副书记、生产经理、总工程师、商务法务经理、质量总监、安全总监、机电经理 |
| 责任担当体 | 核心责任担当体 + 物资经理、财务经理及其他部门负责人等，由核心责任担当体共同确定后在《项目目标管理责任书》中签字 |
| 项目目标管理责任书 | 公司与项目部对项目责任目标的书面约定，是考核奖惩的主要依据 |
| 计划成本（目标成本） | 根据投标资料、合同文件、施工图纸、项目策划书、企业定额、集采价格、公司成本控制指标等资料测算的项目管控目标成本（不含分公司/事业部三级总部的管理成本和项目过程考核、竣工考核兑现奖励） |
| 承接收益率 | （中标价-计划成本）/中标价，按自施部分金额计算 |
| 目标收益率（预收益率） | 在承接收益率基础上，考虑履约创效因素，确定竣工考核时要完成的收益率目标 |

### 职责分工

| 主体 | 职责 |
|------|------|
| 公司商务管理中心 | 负责制度制定、过程督导、考核兑现审批 |
| 分公司/事业部 | 负责责任书签订、过程跟踪、考核实施、奖金分配 |
| 项目部（责任担当体） | 负责目标分解落实、过程履约、配合考核 |
| 人力资源部 | 配合考核、绩效扣发/兑现执行 |
| 财务资金部 | 奖金发放、资金监督 |

## 四维宣贯表

### 做什么（10-15条）

1. 项目开工前签订《项目目标管理责任书》
2. 确定项目目标收益率（预收益率），明确各项责任目标
3. 项目部内部进行目标分解，签订岗位责任书
4. 缴纳风险预留金（项目经理及以上须缴纳）
5. 过程季度跟踪：每季度进行运营分析，检查目标完成情况
6. 过程考核兑现：每季度按节点考核，发放过程奖励
7. 竣工考核兑现：项目竣工后进行最终考核，确认兑现金额
8. 超目标收益分配：按超额累进比例分配奖金（最高49%）
9. 风险预留金返还：按考核结果分批返还（首批50%/二批40%/终额10%）
10. 未完成目标：扣减已发绩效，追责处理
11. 中途项目经理变更：重新签订责任书，责任界定后重新确定目标
12. 亏损/潜亏项目：先行处理原责任担当体成员再重新确定目标
13. 竣工后跟踪：继续跟踪结算、收款、清欠等递延目标

### 怎么做（15-20步）

1. 【分公司/事业部】项目开工后30天内，确定责任担当体成员名单
2. 【分公司/事业部商务部门】起草《项目目标管理责任书》，确定目标收益率
3. 【分公司/事业部总经理】（法定代表人授权委托人）与项目部全体责任担当体成员共同签订责任书
4. 【项目部】项目经理与各岗位签订《经济目标分解岗位责任书》
5. 【责任担当体成员】按规定缴纳风险预留金
6. 【项目部】开工30天内完成目标分解，明确各级责任人、考核时点
7. 【项目部】每月/每季度召开运营分析会，检查目标完成情况
8. 【分公司/事业部】每季度进行过程考核，发放过程奖励
9. 【分公司/事业部】季度考核结果报商务管理中心备案
10. 【项目竣工】分公司/事业部组织竣工考核，出具考核报告
11. 【商务管理中心】审核竣工考核报告，确认超目标收益金额
12. 【财务资金部】计算兑现金额（按超额累进比例，上限49%）
13. 【分公司/事业部】奖金分配方案经责任担当体成员确认
14. 【财务资金部】兑现奖金发放（扣除已发部分）
15. 【项目部】风险预留金按考核结果分批返还（首批50%+二批40%+终额10%）
16. 【项目经理变更】按交接和责任界定情况，重新签订目标责任书
17. 【亏损项目】审计发现问题后，先行处理原责任担当体成员再重新确定目标
18. 【竣工后】继续跟踪结算收款清欠等递延目标完成情况

### 什么时候做

| 时间节点 | 操作内容 | 执行主体 | 成果文件 |
|----------|---------|---------|---------|
| 项目开工前 | 责任担当体确定 | 分公司/事业部 | 责任担当体名单 |
| 开工30天内 | 签订《项目目标管理责任书》 | 分公司总经理+项目部 | 责任书 |
| 开工30天内 | 目标分解，签订岗位责任书 | 项目经理+各岗位 | 岗位责任书 |
| 签约时 | 缴纳风险预留金 | 责任担当体成员 | 缴纳凭证 |
| 每季度 | 过程运营分析会 | 项目部 | 分析报告 |
| 每季度 | 过程考核兑现 | 分公司/事业部 | 考核兑现单 |
| 项目竣工 | 竣工考核，出具考核报告 | 分公司/事业部 | 竣工考核报告 |
| 竣工后 | 审核确认超目标收益 | 商务管理中心 | 审核意见 |
| 竣工后 | 计算兑现金额，发放奖金 | 财务资金部 | 奖金发放单 |
| 竣工后6个月 | 质保金考核兑现 | 分公司/事业部 | 质保金考核单 |
| 风险金首批 | 考核确认后 | 分公司/事业部+财务 | 返还审批单 |
| 风险金二批 | 竣工结算完成后 | 分公司/事业部 | 返还审批单 |
| 风险金终额 | 缺陷责任期满 | 分公司/事业部 | 返还审批单 |
| 项目经理变更 | 工作交接+责任界定+重新签约 | 原项目经理+新项目经理+分公司 | 责任界定书+新责任书 |

### 怎么考核

- **考核主体**：分公司/事业部（商务部门牵头）、商务管理中心、人力资源部、财务资金部
- **考核方式**：目标完成度审计、过程检查、竣工综合考核
- **考核周期**：季度过程考核 + 竣工最终考核
- **超目标收益兑现比例**（累进）：
  - 超目标收益率 1% → 兑现比例 31%（即30%+1%）
  - 超目标收益率每增加1% → 兑现比例增加1%
  - 兑现比例上限为 **49%**
- **兑现分配**：
  - 按个人风险预留金缴纳比例及贡献情况分配
  - 未缴纳风险预留金人员不参与分配
- **未完成目标**：
  - 扣减已发放的绩效薪
  - 追责处理（项目经理为第一终身责任人）
  - 亏损项目风险预留金不予返还

## 核实要点清单

### 资料清单

1. 《项目目标管理责任书》（双方签字盖章）
2. 《风险预留金缴纳凭证》（各责任担当体成员）
3. 《连带责任担保书》
4. 《经济目标分解岗位责任书》（各岗位）
5. 季度运营分析报告（每季度）
6. 过程考核兑现审批单（每季度）
7. 竣工考核报告（分公司/事业部出具）
8. 商务管理中心审核意见
9. 奖金分配方案及确认签字
10. 奖金发放凭证
11. 风险预留金返还审批单（分三批）
12. 项目经理变更时的责任界定书
13. 重新签订目标责任书的审批记录
14. 亏损/潜亏项目的中期审计报告
15. 递延目标完成情况跟踪记录

### 常见问题/风险点

1. **目标收益率设定不合理**：承接收益率低于成本能力 → 对照中标价与计划成本核查计算是否准确
2. **风险预留金未足额缴纳**：责任担当体成员未按时足额缴纳 → 核查缴纳凭证与责任书约定金额是否一致
3. **过程考核流于形式**：未实际跟踪目标完成情况 → 核查季度分析报告是否真实反映问题
4. **项目经理变更未重新签约**：变更后未及时重新确定目标 → 核查变更时间和重新签约时间节点
5. **奖金分配不透明**：未按风险预留金缴纳比例分配 → 核查分配方案是否有责任担当体全体签字确认
6. **亏损项目追责不到位**：项目亏损但未进行责任追究 → 核查审计报告和责任认定记录
7. **风险金返还违规**：未达条件提前返还 → 核查三批返还条件是否分别满足
8. **连带担保未落实**：未签订担保书或担保人不足三人 → 核查担保书原件及担保人资质

### 关键指标

| 指标 | 计算方式 | 达标标准 |
|------|---------|---------|
| 目标收益率完成率 | 实际收益率/目标收益率×100% | ≥100% |
| 风险预留金缴纳及时率 | 按时缴纳次数/应缴纳次数×100% | 100% |
| 过程考核兑现执行率 | 已兑现次数/应兑现次数×100% | 100% |
| 竣工考核及时率 | 竣工后3个月内完成考核次数/应考核次数×100% | 100% |
| 奖金分配合规率 | 合规分配次数/总分配次数×100% | 100% |
| 责任书及时签订率 | 按期签订次数/应签订次数×100% | 100% |
| 目标分解完成率 | 按期完成目标分解次数/应分解次数×100% | 100% |
