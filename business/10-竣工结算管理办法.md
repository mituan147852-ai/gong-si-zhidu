## 四维宣贯表

### 做什么

- **目的**：坚持管控要素清晰、责任目标明确、奖罚分明，提升企业盈利能力
- **适用范围**：公司职能部门、分公司/事业部、项目部
- **管理原则**：管控要素清晰、责任目标明确、奖罚分明、提升盈利能力
- **"21162"核心目标**：
  - **2**（两次策划）：项目竣工验收或交付前2个月内完成项目成本预估、结算资料整理、收益分析，结算策划通过公司评审
  - **1**（一份责任状）：竣工验收或交付前1个月完成结算责任书签订
  - **1**（一次报出）：竣工验收或交付后1个月内报出结算书
  - **6**（六个月完成）：力争竣工或交付后6个月内完成竣工结算
  - **2**（两个月复盘）：竣工结算后2个月内完成考核兑现、复盘管理
- **主要应对风险**：未及时结算风险、亏损风险、未达目标收益风险

### 怎么做

1. **结算报出阶段**：
   - **结算策划**（竣工前60天）：项目经理作为第一责任人组织技术总工、商务经理、生产经理等进行
   - **结算初稿编制**：汇总合同内重计量、已确权签证/变更/索赔、结算策划事项等资料
   - **成本锁定**（竣工前60天）：汇集实施过程所发生成本，进行成本锁定和收益分析；不达收益项目15天内完成阶段成本复盘并上报公司评审
   - **结算书编制**（竣工后30天内）：完成结算书组卷报出
2. **结算评审、提交和定案**：
   - **评审**（竣工前30天）：分公司/事业部商务部组织结算书和结算策划评审并提出整改意见
   - **调整**：项目部根据整改意见调整结算书和结算策划
   - **目标奖罚责任状签订**（竣工前1个月）
   - **提交**：取得发包人收件人书面签收文件
   - **审核**（竣工后6个月内完成）
   - **定案**：须经项目经理、分公司/事业部总经济师及总经理、商务中心主管副主任、总经济师评审签字盖章
3. **成本复盘**（竣工结算完成后2个月内）
4. **重大/久竣未结项目**：由两级总部成立专项结算推进工作小组负责推进

### 什么时候做

| 事项 | 时间节点 |
|------|---------|
| 结算策划启动 | 竣工验收/交付前60天 |
| 成本锁定及收益分析 | 竣工验收/交付前60天 |
| 不达收益项目成本复盘上报 | 15天内完成 |
| 结算书和策划评审 | 竣工验收/交付前30天 |
| 结算书组卷报出 | 竣工验收/交付后30天内 |
| 结算责任状签订 | 竣工验收/交付前1个月 |
| 完成竣工结算 | 竣工/交付后6个月内 |
| 结算定案审批 | 结算审核完成后 |
| 成本复盘 | 竣工结算完成后2个月内 |
| 季度推进分析会 | 每季度 |

### 怎么考核

- **考核主体**：项目部（结算策划和结算书编制）、分公司/事业部（结算策划评审、结算报出评审、定案评审）、公司商务管理中心（备案、审核）、总经济师（审批）
- **考核方式**：责任状考核；每季度公司组织竣工项目结算推进分析会
- **激励约束措施**：按《竣工项目结算奖罚管理办法》执行
- **重大结算项目**：结算自施金额超过10亿元；久竣未结项目（竣工超2年）由两级总部成立专项推进小组
