## 四维宣贯表

### 做什么

- **目的**：建立标准化、预控策划、动态监控的商务风险管理体系，及时识别和化解项目商务风险
- **适用范围**：公司职能部门、分公司/事业部、项目部
- **管理原则**：标准化管理、预控策划、动态监控
- **主要应对风险**：运营管理不到位风险、成本/收益风险、结算风险
- **重点监控项目类型**：集团级商务重点项目、成本/收益风险项目、过程结算风险项目、竣工结算逾期项目、停工风险项目、主合同触限制性事项风险项目

### 怎么做

1. **重点监控项目确定**：根据集团级商务重点项目、成本风险（红色：履约过程实际收益率<0）、过程结算风险、竣工结算风险、停工风险、主合同触红线等原则筛出重点监控项目
2. **风险分级**：对各类风险项目按黄/橙/红色进行分级预警（见"什么时候做"）
3. **动态管理**：每月末根据各项风险筛出公司风险项目，根据具体情况增减
4. **三级管控**：实行项目部→分公司/事业部→公司总部递增式三级管控
5. **风险化解跟踪**：对风险项目进行过程跟踪、措施落地、风险化解
6. **重大决策**：需决策的风险项目上报公司风险合规委员会；连续六个月未化解的重大潜亏项目须召开两级总部专题会议决策是否成立风险化解小组

### 什么时候做

| 风险类别 | 风险等级 | 标准 |
|---------|---------|------|
| 成本/收益风险 | 红色 | 履约过程中实际收益率＜0 |
| 过程结算风险 | 黄色 | 过程结算率<90% |
| 过程结算风险 | 橙色 | 过程结算率<80% |
| 过程结算风险 | 红色 | 过程结算率<75% |
| 过程结算风险 | 红色 | 已完工未结算金额/自施合同额＞10%或连续三月以上且金额＞1000万 |
| 竣工结算风险 | 黄色 | 竣工后1年以上未满1.5年未完成结算 |
| 竣工结算风险 | 橙色 | 竣工后1.5年以上未满2年未完成结算 |
| 竣工结算风险 | 红色 | 竣工超2年未完成结算 |
| 停工风险 | 黄色 | 停工时长＜3个月 |
| 停工风险 | 橙色 | 3个月≤停工时长＜6个月 |
| 停工风险 | 红色 | 停工时长≥6个月 |
| 主合同风险 | 红色 | 触及A类限制性事项 |

**管理节点**：重点监控项目动态管理每月末更新；触A类限制性事项按底线管理办法执行；连续六个月未化解须开会决策

### 怎么考核

- **考核主体**：分公司/事业部（整理重点监控项目清单、跟踪落实风险化解）、公司商务管理中心（监控、跟踪落实）
- **考核方式**：月度动态跟踪管理；季度更新重点监控项目清单
- **合规要求**：触限制性事项A类事项在监控过程中出现"未落实"情况，按底线管理办法执行
