## 核心条款

### 目的和依据
> 为深入贯彻落实习近平新时代中国特色社会主义思想和党的二十大精神，贯彻新发展理念，促进高质量发展，结合公司改革发展的新思路、新目标、新举措，贯彻落实中建一局全面开展精细化管理的精神，实现降本增效的目的。根据中建一局和公司有关规章制度，制定本细则。

### 适用范围
> 本细则适用于分公司（事业部）和项目部的境内外房建工程、基础设施工程的采购工作。

### 管理原则
> **采购集中、项目执行原则。** 采购业务组建设按照该原则开展工作——集中采购权归业务组，项目层面负责执行落地。

### 职责分工
| 角色 | 职责定位 |
|------|------|
| **商务管理中心**（公司层面） | 主管：指导、协助、督促分公司建立采购业务组；监督、检查、考核运行情况；对采购成果统计分析和发布；负责人员培训 |
| **分公司（事业部）商务法务部** | 主管：监督、检查采购业务组人员工作情况；负责人员考核和奖罚；协调采购业务组与项目商务人员的分工 |
| **采购业务组** | 执行主体：牵头集采、组织招标、审核计划、管理分供商资源库、执行合规管理 |
| **项目部** | 配合执行：编制招采计划、参与集采、反馈结果、执行中标结果、负责项目单项招标（提级之外） |

### 主要应对的风险
1. **采购计划风险**：缺乏科学合理的计划，市场变化趋势预测不准确，与采购目标存在较大偏差。
2. **成本控制风险**：缺乏成本有效控制或降低成本方法，导致采购价格大幅超出市场平均水平。
3. **合同履约风险**：对分供商的资信、资金等情况调查不深入，在合同履约过程中出现纠纷。
4. **采购合规风险**：采购活动中存在不合规行为导致中国建筑品牌、形象受损。
5. **廉洁管理风险**：管理控制不到位，采购核心环节缺少监控，可能导致采购不合规，损害公司利益。

## 四维宣贯表

### 做什么（14条）

1. 牵头或参与股份、一局、公司集采工作，确保分公司（事业部）推荐资源中标率。
2. 牵头组织、实施、协调、落实分公司（事业部）集采工作，研究解决集采中的问题，协调资源整合，推进集采工作。
3. 负责集采计划编制、招标组织和任务创建，确定集采价格及单位并发布集采中标成果（包括编制发布招标公告及招标文件、组织相关人员投标、开标和评标、定标、发布中标通知、编制合同文件等）。
4. 负责实时掌握项目分包分供价格，对不符合市场行情的价格及时调整。
5. 按照公司要求监督项目执行股份、一局、公司集采成果执行应用，审核应执行未执行情况。
6. 负责定期组织召开分公司（事业部）集采工作会、交底会、复盘会。
7. 负责所辖项目单项招标提级采购工作（包括编制发布招标公告及招标文件、组织相关人员投标、开标和评标、定标、发布中标通知、编制合同文件等）。
8. 负责审核所辖项目《项目整体资源需求招采计划》《项目前期招采计划》《项目年度招采计划》，并按制度要求报公司备案。
9. 按照项目招采计划开展采购工作，并督导招采计划执行。
10. 对新引进的分供商进行考察评价，按照《中建一局集团第二建筑有限公司资源管理办法》建立本区域分供商资源库，对本区域分供商进行管理。
11. 按照上级单位要求完成智采商城相关管理工作。
12. 负责云采集采平台管理及分公司（事业部）采购合规管理工作。
13. 编制《项目整体资源需求招采计划》《项目前期招采计划》《项目年度招采计划》，并发起招采申请（项目部职责）。
14. 负责项目单项招标提级采购之外招标任务，包括编制发布招标公告及招标文件、组织相关人员投标、开标和评标、定标、发布中标通知、编制合同文件等（项目部职责）。

### 怎么做（详细操作流程，16步）

| 步骤 | 涉及部门/岗位 | 具体操作 | 记录报表 | 签字审批 |
|------|------|----------|----------|----------|
| **第1步** | 项目部 | 编制《项目整体资源需求招采计划》《项目前期招采计划》《项目年度招采计划》，发起招采申请 | 招采计划文件 | 项目经理签字 |
| **第2步** | 采购业务组 | 审核项目招采计划，汇总分公司（事业部）所辖项目生产资源需求，识别常规、量大、通用资源 | 审核记录 | 采购业务组负责人签字 |
| **第3步** | 采购业务组 | 将汇总结果按制度要求报公司备案（审核通过后） | 备案表 | 商务法务部经理签字 |
| **第4步** | 采购业务组 | 根据集采安排，牵头组织或参与股份/一局/公司集采工作，编制集采计划 | 集采计划文件 | 分公司总经济师审批 |
| **第5步** | 采购业务组 | 编制发布招标公告及招标文件（含采购需求、评标标准、合同条款） | 招标公告、招标文件 | 商务法务部经理审核 |
| **第6步** | 采购业务组 | 组织相关人员投标，开展开标和评标工作 | 开标记录、评标报告 | 评标小组签字 |
| **第7步** | 采购业务组 | 定标，确定集采价格及单位，发布中标通知 | 中标通知书 | 分公司总经济师审批 |
| **第8步** | 采购业务组 | 编制合同文件，组织中标单位签订合同 | 采购合同 | 法务审核+总经济师签章 |
| **第9步** | 采购业务组 | 实时掌握项目分包分供价格，与市场行情比对，对异常价格及时调整 | 价格台账 | 采购业务组负责人 |
| **第10步** | 采购业务组 | 监督项目执行集采成果应用，审核应执行未执行情况，督促整改 | 监督检查记录 | 商务法务部经理签字 |
| **第11步** | 采购业务组 | 对新引进分供商进行考察评价，纳入本区域分供商资源库，对在用分供商进行动态评价 | 考察评价报告、分供商台账 | 商务法务部经理审核 |
| **第12步** | 采购业务组 | 组织召开分公司（事业部）集采工作会、交底会、复盘会，形成会议纪要 | 会议纪要 | 与会人员签字 |
| **第13步** | 采购业务组 | 负责所辖项目单项招标提级采购（10万以上物资/设备租赁、30万以上专业分包/服务、劳务分包等），全流程执行招标公告→投标→开标→评标→定标→发中标通知→签合同 | 提级采购全流程文件 | 分公司总经济师审批 |
| **第14步** | 采购业务组 | 完成智采商城相关管理工作，确保线上采购流程合规 | 智采商城操作记录 | 采购业务组负责人 |
| **第15步** | 商务管理中心 | 对分公司采购业务组工作成果进行统计、分析和发布，按季度检查指标完成情况 | 统计报告、考核报告 | 商务管理中心负责人 |
| **第16步** | 商务法务部 | 对采购业务组人员工作情况进行监督、检查、考核和奖罚，协调与项目商务人员的分工 | 考核记录、奖罚通知 | 商务法务部经理 |

### 什么时候做

| 时间节点 | 操作内容 | 执行主体 | 成果文件 |
|------|------|------|------|
| **常年/实时** | 实时掌握项目分包分供价格，对不符合市场行情的价格及时调整 | 采购业务组 | 价格台账 |
| **常年/实时** | 执行集采、联条工作，监督集采成果在项目执行中的应用 | 采购业务组、项目部 | 监督检查记录 |
| **常年/实时** | 智采商城相关管理工作及云采集采平台管理 | 采购业务组 | 平台操作记录 |
| **常年/实时** | 分供商资源库动态管理（新引进考察评价、在用分供商动态评价） | 采购业务组、项目部 | 考察评价报告、分供商台账 |
| **每月** | 组织召开分公司（事业部）集采工作会、交底会、复盘会 | 采购业务组 | 会议纪要 |
| **季度** | 公司按季度检查指标完成情况，督导完成 | 商务管理中心 | 考核报告 |
| **季度/不定期** | 公司开展对分公司（事业部）采购合规性抽查 | 商务管理中心 | 抽查报告 |
| **项目启动阶段** | 项目部编制《项目整体资源需求招采计划》《项目前期招采计划》，发起招采申请 | 项目部 | 招采计划文件 |
| **年度** | 项目部编制《项目年度招采计划》，采购业务组审核并报公司备案 | 项目部、采购业务组 | 年度招采计划、备案表 |
| **集采周期内** | 按上级单位集采安排，完成集采计划编制→招标公告发布→投标→开标评标→定标→发布中标通知→合同签订全流程 | 采购业务组 | 集采全流程文件 |
| **提级采购发生时** | 对单项提级采购范围内的业务（10万+物资/设备租赁、30万+专业分包/服务、劳务分包），执行单项招标提级采购全流程 | 采购业务组 | 提级采购全流程文件 |

### 怎么考核

| 考核维度 | 具体内容 |
|----------|----------|
| **考核主体** | 商务管理中心（公司层面）+ 商务法务部（分公司层面） |
| **考核方式** | 定期检查（季度）+ 不定期抽查（合规性）；台账记录、报告文件审查 |
| **考核周期** | 季度检查指标情况；日常监督采购合规性 |
| **考核内容** | 指标完成情况 + 采购合规性 |
| **奖惩机制** | 由商务法务部负责对采购业务组人员进行考核和奖罚（具体奖惩标准按公司绩效考核制度执行） |
| **不合格情形** | 集采成果应执行未执行、采购不合规、价格大幅偏离市场行情、分供商调查不深入导致合同纠纷等 |
| **结果应用** | 考核结果纳入人员绩效；违规行为按公司制度追究责任 |

### 常见问题/风险点（7条）

1. **集采成果执行不到位**：项目未按规定执行股份/一局/公司集采结果，需建立应执行未执行审核机制，采购业务组须督导整改并留存记录。
2. **采购价格偏离市场**：缺乏价格动态监控，价格与市场行情偏差过大未及时调整，引发成本控制风险——须定期更新价格台账并比对市场。
3. **分供商资信调查不足**：对新引进分供商未进行充分考察评价，履约过程中出现纠纷——须严格执行考察评价流程并形成书面报告。
4. **合同履约纠纷**：合同条款与分供商实际履约能力不匹配，调查不深入——须在合同签订前完成资信、资金情况核实。
5. **提级采购范围界定模糊**：未准确判断采购是否属于提级范围，导致拆分规避招标——须对照制度明确标准（10万+物资/设备租赁、30万+专业分包/服务、劳务分包）。
6. **采购合规性缺失**：采购核心环节缺少监控，存在廉洁风险——须强化全过程监督，关键节点多人审核会签。
7. **招采计划编制滞后**：项目部未按时编制并提交招采计划，影响集采统筹安排——须在项目启动阶段和年度节点按时完成并备案。

### 关键指标

| 指标类别 | 指标名称 | 说明 |
|----------|----------|------|
| **集采执行** | 推荐资源中标率 | 采购业务组推荐资源在集采中的中标比例，反映资源整合能力 |
| **成本控制** | 采购价格偏离度 | 采购价格与市场行情的偏差幅度，偏离过大概括成本控制风险 |
| **合规管理** | 采购合规率 | 合规采购笔数占采购总笔数的比例，合规性抽查结果为重要依据 |
| **计划执行** | 招采计划完成率 | 实际完成招采数量占计划招采数量的比例，反映计划执行力度 |
| **供应商管理** | 分供商考察覆盖率 | 已完成考察评价的分供商数量占在用分供商总数的比例 |
| **平台管理** | 智采商城应用率 | 纳入智采商城管理的采购业务占比，反映线上采购覆盖率 |
