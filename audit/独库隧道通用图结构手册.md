## 一、图纸编号体系

本册图纸采用 **TYSD-X-Y-Z** 三级编号体系：X=专业篇号（1~13），Y=分项序号，Z=子序号。

### 编号前缀含义

| 前缀 | 含义 | 示例 |
|------|------|------|
| `KS-` | 进口（进口端）Ⅴ级围岩洞口衬砌 | KS-Ⅴa、KS-Ⅴb |
| `XKS-` | 斜井进入（进口方向）Ⅴ级围岩 | XKS-Ⅴa、XKS-Ⅴb |
| `KⅨS-` | Ⅸ合同段斜井进入（进口方向） | KⅨS-Ⅴa、KⅨS-ⅤQa |
| `XKⅨS-` | Ⅸ合同段斜井进入（出口方向） | XKⅨS-Ⅴa |
| `KJS-` | Ⅸ合同段进口（洞口明洞段） | KJS-Ma、KJS-Ⅴa |
| `S-` / `XS-` | 服务隧道（主洞/斜井出口方向） | S-Ⅴa、XS-Ⅴa |
| `SC-` | 西服务隧道入口段（Service tunnel Cross） | SC-Ⅴ、SC-Ⅳ |
| `SX-` | 西服务隧道出口段 | SX-Ⅴ、SX-Ⅳ |
| `SJ-` | 紧急停车带（Service road Junction） | SJ-Ⅴa、SJ-Ⅳb |

| `SYB-` | 岩爆段 | SYB-Ⅰ~Ⅳ |
| `SDBX-` | 软岩大变形段（软大变） | SDBX-Ⅰ~Ⅲ |

### 围岩等级后缀

| 后缀 | 含义 |
|------|------|
| `Ma` / `Mb` | Ma=严重偏压洞口明洞；Mb=无偏压/轻微偏压明洞 |
| `Q`（加强型） | Qu강=加强，Qa/Qb=加强亚型；Q=Ⅴ级断层影响段 |
| `QDF`（强断层） | QDF=强断层破碎带（含底拱）；QD=断层影响段 |
| `a / b / c / d` | 支护参数变型；d有时表示单心圆断面 |
| `Ⅰ / Ⅱ / Ⅲ / Ⅳ` | SYB/SDBX特殊围岩细分等级，等级越高地质条件越差 |

### 每套衬砌图册的标准组成（四件套）

1. **结构设计图**（1页）：衬砌轮廓、总净空、混凝土等级（常用C30/C35）、厚度参数
2. **钢支撑设计图**（1~3页）：I20a/I22a型钢钢架或φ25格栅，间距0.6~1.0m，连接钢筋
3. **配筋设计图**（1页）：主筋φ22~25@200，箍筋φ8@150~200，分布筋φ12~16
4. **超前支护方案**（TYSD-6系列）：φ42超前小导管或φ108长管棚，注浆参数

## 二、建筑限界及内轮廓（TYSD-2）

建筑限界是隧道设计的"刚性边界"，规定隧道净空不得侵入的最小尺寸。所有后续衬砌内轮廓均以其为基础。

### 限界分类

- **主洞**：Ⅴ/Ⅳ/Ⅲ级内轮廓各一套，Ⅴ级净空最大（考虑钢架侵占空间）
- **紧急停车带**：每750~1500m设置一处，单侧布置
- **车行横通道**：连接左右洞，供救援车辆通行，间距≤600m
- **配电横通道**：仅供机电管线通过，尺寸最小
- **人行横通道**：仅供人员通行，间距≤500m
- **服务隧道**：位于双洞之间，作为应急救援主通道，宽度6.0m

## 三、明洞衬砌结构（TYSD-3-1）

明洞（Open Cut Tunnel）指洞口段采用明挖法施工、回填后形成覆盖层的隧道结构。用于浅埋偏压洞口或地形限制无法暗挖的地段。

| KⅨS-Mb | TYSD-3-1-7 | 页17 | TYSD-3-1-8 | 页18~19 | 轻微偏压 |

### 明洞与暗洞的区别

- **明洞**（KS/KⅨS/KJS）：明挖施工，回填土压力承担主要荷载，衬砌主要承受土压力+回填荷载，厚度较大（60~100cm），配筋率高
- **暗洞**（标准Ⅴ/Ⅳ/Ⅲ级衬砌）：钻爆法暗挖，围岩与支护共同承载，衬砌作为安全储备，Ⅴ级厚度约45~50cm

### 偏压判断

- **Ma（严重偏压）**：地面横坡>1:1.25（约44°）；埋深<2倍洞跨；洞顶覆盖层单侧显著偏薄
- **Mb（无/轻微偏压）**：地面横坡<1:2（约27°）；埋深≥2倍洞跨；洞顶覆盖层基本对称

### ⅤQa/ⅤQb vs 普通Ⅴa/Ⅴb 的区别

Q（Qu강/加强）系列用于：
- 位于发震断裂带影响范围内（地震加速度峰值>0.2g）
- 地质条件差于普通Ⅴ级（断层破碎带、涌水段）

### Ⅳb/Ⅳc为何需要表面防裂钢筋？

Ⅳb/Ⅳc型衬砌的曲率半径和截面厚度组合，在混凝土硬化过程中因温度收缩和干燥收缩产生表面拉应力，容易导致环向裂缝。表面防裂钢筋（φ8~10@150×150mm钢筋网，位于衬砌表面5cm处）的作用是**控制表面裂缝开展，将裂缝宽度限制在0.2mm以内**。

### Ⅳd单心圆断面

单心圆（Single Center）指仰拱与拱部采用同一曲率半径的圆弧连接，与常见的三心圆不同。适用于围岩条件相对稳定、无底鼓风险的Ⅳ级围岩段落。

### Ⅳ级围岩每延米工程量参考

- **混凝土**：C30 ~0.7~0.9m³/m
- **钢筋**：60~90kg/m（含钢架折算）
- **钢支撑**：格栅/I18a，间距0.75~1.0m
- **锚杆**：L=3.0m，约6~8根/m
- **防裂钢筋网**：φ8@150×150，仅Ⅳb/Ⅳc设置

### 一、揭煤防突及瓦斯治理（页398~406）

**四阶段流程**：
1. **区域预测**：施工前通过钻探确定煤层位置、厚度、瓦斯压力
2. **工作面预测**：距煤层20m前开始；钻屑指标法（钻屑量S、钻屑瓦斯解析指标K1）
3. **超前抽放**：φ75~100mm抽放孔，间距2~3m，覆盖整个煤层范围；抽放负压≥13kPa
4. **揭煤程序**：远距离放炮揭煤；通风≥0.5m/s；瓦斯浓度<0.5%方可恢复施工

## 二十三、围岩选型逻辑与快速查找

### 六步定位法

**Step 1** — 判断隧道类型（哪个位置？）
→ 主洞 → TYSD-3-2/3-3 | 明洞 → TYSD-3-1 | 紧急停车带 → TYSD-4-1 | 服务隧道入口段 → TYSD-4-2 | 配电横通道 → TYSD-4-3 | 支洞 → TYSD-4-6

**Step 2** — 判断围岩等级（Ⅴ/Ⅳ/Ⅲ）
→ Ⅴ级：钢支撑必设，超前支护必做 | Ⅳ级：配钢支撑，局部超前 | Ⅲ级：一般无钢支撑

**Step 3** — 判断洞口方向（哪个系列？）
→ KS-=进口 | XKS-=斜井进入 | KⅨS/XKⅨS=Ⅸ合同段 | KJS=Ⅸ合同段进口 | S/XS=服务隧道 | SJ=紧急停车带 | SD=配电横通道

**Step 4** — 判断特殊条件
→ 偏压（Ma/Mb）→ 明洞专用 | 断层（Q/QD/QDF）→ 服务隧道Ⅴ级用 | 抗震加强（Qa/Qb）→ 地震设防区

**Step 5** — 确认结构图编号
→ 例：KⅨS-ⅤQa型结构图=TYSD-3-2-17（页40）

**Step 6** — 匹配配套图纸
→ 每套衬砌 = 结构图 + 钢支撑图 + 配筋图 + 超前支护方案（四件套）
