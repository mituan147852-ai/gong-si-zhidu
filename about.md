# 宣贯格式说明

## 四维宣贯表

每个制度配套以下四个维度，帮助快速理解和执行：

| 维度 | 说明 |
|------|------|
| **做什么** | 制度的核心目标和工作内容 |
| **怎么做** | 具体的操作流程和方法步骤 |
| **什么时候做** | 时间节点和节奏把控 |
| **怎么考核** | 考核主体、方式、奖惩标准 |

---

## 状态标识

| 标识 | 含义 |
|------|------|
| 🟢 已发布 | 现行有效，正在执行 |
| 🟡 修订中 | 正在修订，暂沿用旧版 |
| 🔴 已废止 | 不再执行 |

---

## 制度分类

- **财务资金制度**：资金管理、费用管理、资产管理等
- **商务制度**：目标考核、成本管理、供方管理、分包结算等
- **项目管理部制度**：前期手续、项目实施、竣工保修等
