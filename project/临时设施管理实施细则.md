## 四维宣贯表

### 做什么

- **目的**：本着"降本增效"的管理原则，在确保临时设施使用功能和使用安全的基础上，进一步规范现场临时设施的建造标准和管理流程。
- **适用范围**：公司所属全部工程项目。
- **管理原则**：标准化原则、经济性原则、手续规范化原则；确保临时设施"科学合理、功能齐全、经济适用、安全便捷、快速装配、高效周转"。
- **主要应对风险**：规模超标风险、采购不规范风险、建造不合格风险、使用不当风险、维护不到位风险、安全消防风险、环保卫生风险、处置不当风险、账实不符风险。
- **管理内容**：临时设施配置标准（分级A/B/C）、方案管理、采购管理、施工验收管理、维护管理、过程拆改管理、成本归集和摊销、资产入账管理、盘点、周转使用、调拨和处置。

### 怎么做

1. **分级配置标准**：
   - A级（示范）：国内行业领先水平，适用于业主合同要求高、创国家级奖项目、公司重点打造项目
   - B级（优秀）：区域领先水平
   - C级（达标）：国内行业平均水平，标准项目临时设施配置，费用不超过合同额对应比例
2. **方案管理**：临时设施应单独编制《临时设施专项施工方案》《临时用水专项施工方案》《临时用电专项施工方案》，按照《施组方案管理办法》履行策划、编制、审核、审批、交底流程；方案审批前须完成采购合同招标。
3. **采购管理**：一局战略采购范围内的按战略采购执行；其他按公司招标采购管理办法执行；严禁未签合同进场施工。
4. **施工验收**：临时设施施工标准及做法依据《中建一局集团施工现场标准化图册》执行；施工完成后3天内由项目经理组织验收；A/B类临时设施总部及分公司/事业部全程参与验收；一个月内完成结算及资产验收。
5. **维护管理**：每月全面检查一次；使用年限超过5年的临时设施应进行防火性能检测。
6. **成本归集与摊销**：自行建造费用通过"在建工程"归集；按工期摊销，可重复使用的以5年为折旧年限；按月计提折旧。
7. **周转使用**：分公司/事业部范围内年度临时设施周转使用率应不低于30%，纳入年度考核指标。
8. **调拨和处置**：项目结束后由项目部编制临时设施处置计划，经审批后实施；处置方式包括调拨、报废、出售；严禁未经审批私自处置。

### 什么时候做

| 节点 | 内容 |
|------|------|
| 采购合同招标前 | 完成临时设施方案审批 |
| 施工完成后3天内 | 完成临时设施实体验收 |
| 建造完成后1个月内 | 完成结算及资产验收 |
| 建造完成后 | 项目部进行费用核算，形成书面核算报告 |
| 每月 | 项目部对临时设施进行全面检查 |
| 每年 | 临时设施实地盘点及价值评估（至少一次） |
| 每季度 | 分公司/事业部对所辖项目临时设施使用情况检查；盘点临时设施周转率 |
| 每年12月31日前 | 完成年度临时设施周转率统计上报 |

### 怎么考核

- **考核主体**：公司项目管理部（归口管理）、商务管理部、科技质量部、机电工程事业部、财务资金部、党建工作部、安全生产监督管理部、企业策划部/信息部；分公司/事业部；项目部。
- **考核方式**：临时设施验收检查、项目部定期检查、分公司/事业部定期检查、公司总部抽查（抽查比例不小于年度新建临时设施的30%）。
- **奖惩措施**：
  - 违反采购规定：视情况主要责任人罚款1000-5000元
  - 未经批准擅自拆改：视情况项目经理罚款3000-5000元
  - 费用超标：超出预算10%以内罚3000元；10%-20%罚4000元；20%以上罚5000元
