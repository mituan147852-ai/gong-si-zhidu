## 四维宣贯表

### 做什么（要点充沛，8-15条）

1. **制定履约品质评价方案**：公司总部每年制定履约品质评价方案，明确提升目标和具体提升措施。
2. **季度履约品质评价**：分公司/事业部按季度对所辖在施项目进行履约品质评价，形成评价结果报公司项目管理部备案。
3. **半年履约品质综合评价检查**：公司组织对项目进行履约品质综合评价检查，上半年必须于5月31日前完成，下半年必须于11月30日前完成。
4. **评价内容七维度**：技术质量管理（25%）、进度管理（18%）、安全管理（15%）、环保管理（10%）、客户服务管理（15%）、CI形象管理（7%）、劳务管理（10%），合计100%。
5. **评价等级划分**：A类（≥95分）、B类（85-95分）、C类（75-85分）、D类（<75分）。
6. **等级降级情形**：碰触8类情形之一→原等级降一级；碰触6类情形之一→原等级降两级。
7. **A类项目奖励**：奖励项目部20万元（不纳入项目成本）。
8. **C/D类项目处置**：连续两次C类→通报批评+约谈项目经理；D类→项目经理免职处置，3年内不得担任项目经理，不良记录录入业绩档案。
9. **负面警示培训**：每年年中组织C/D类项目项目经理等人员参加2-3天封闭培训，费用个人负担，不合格者免职。
10. **检查整改**：检查组成员现场下达整改单，项目部规定时间内完成整改并书面回复。
11. **底线处罚**：遇违反底线管理规定情况，现场下达底线处罚通知单，必要时可下达停工令。
12. **客户满意度管理**：协助项目及时处理违法违规事件，减轻和消除对企业品牌的影响。
13. **观摩活动创建**：事业部/分公司指导项目积极承办大型观摩活动，创建属地履约奖项，为企业信用加分。
14. **专项提升培训**：对D类项目/检查组后五名/重大隐患项目组织履约品质专项提升培训。
15. **履约品质预警和底线项目处置**：对评价结果中的预警项目和底线项目进行处置。

### 怎么做（操作流程详细展开，10-20步）

#### 一、履约品质评价流程

**步骤1：制定年度评价方案**
- 发起部门：项目管理部
- 操作动作：每年制定履约品质评价方案，明确当年提升目标和具体提升措施
- 各业务系统负责结合年度工作要点和管控重点对评分标准进行调整，于考核前发布
- 输出文件：年度履约品质评价方案

**步骤2：季度履约品质评价（分公司/事业部）**
- 发起部门：分公司/事业部
- 操作动作：按季度对所辖在施项目进行履约品质评价
- 评价内容：七维度（技术质量25%/进度18%/安全15%/环保10%/客户服务15%/CI形象7%/劳务10%）
- 各业务系统修订本系统评价标准，对检查人员进行交底培训，统一检查尺度
- 输出文件：评价结果、履约品质检查报告
- 报送至：项目管理部

**步骤3：半年履约品质综合评价（公司总部）**
- 发起部门：项目管理部
- 操作动作：组织各业务部门对项目进行综合评价检查
- 按项目类型分组：房建组（按区域划分两组）/基础设施组/海外工程组
- 检查组成员由各业务部门指定，要求具备专业能力和职业道德
- 输出文件：综合评价报告

**步骤4：现场检查**
- 发起部门：检查组
- 操作动作：
  - 充分发掘项目管理亮点、剖析项目存在的问题，收集相关资料或影像证明作为评价依据
  - 现场下达整改单（规定日期内完成整改并书面回复）
  - 遇违反底线管理规定→现场下达底线处罚通知单
  - 必要时→下达停工令
- 输出文件：整改通知单、底线处罚通知单、停工令（如有）

**步骤5：检查总结会**
- 发起部门：项目管理部
- 操作动作：项目履约品质检查结束后召开总结会
- 各业务系统汇报系统排名及检查情况
- 对本系统排名前10%及后10%的项目进行重点阐述
- 输出文件：会议纪要

**步骤6：评价结果定级**
- 发起部门：项目管理部
- 操作动作：
  - 按综合得分确定基础等级：A类≥95分/B类85-95分/C类75-85分/D类<75分
  - 核查是否碰触降级情形（8类降一级/6类降两级）
  - 最终确定项目履约品质类别
- 输出文件：评价等级汇总表

**步骤7：A类项目奖励**
- 发起部门：项目管理部
- 操作动作：A类项目→奖励项目部20万元（不纳入项目成本）
- 输出文件：奖励文件

**步骤8：C/D类项目处置**
- 发起部门：项目管理部
- 操作动作：
  - 连续两次C类→公司范围内通报批评，由公司履约副总经理约谈项目经理
  - D类→项目经理免职处置，公司范围内通报批评，3年内不得担任项目经理，不良记录录入业绩档案
- 输出文件：处置文件/通报

**步骤9：负面警示培训组织**
- 发起部门：项目管理部
- 操作动作：
  - 每年年中组织一次负面警示培训
  - 对象：C/D类项目项目经理、生产经理、项目总工、质量总监、安全总监
  - 封闭培训2-3天，期间不允许请假及中途离场
  - 培训费及食宿等费用由参加培训人员个人负担
  - 培训考试不合格人员→免职处置
- 输出文件：培训通知、参训名单、考核结果

#### 二、检查结果申诉流程

**步骤1：整改实施**
- 发起部门：项目部
- 操作动作：收到整改单后在规定日期内完成整改，形成书面回复
- 输出文件：整改回复

**步骤2：申诉（如有）**
- 发起部门：项目部
- 操作动作：对评价结果有异议，向项目管理部提出申诉
- 输出文件：申诉书

**步骤3：申诉处理**
- 发起部门：项目管理部
- 操作动作：受理并审核申诉，必要时组织复查
- 输出文件：申诉处理意见

### 常见问题 / 风险点

1. **评分标准不统一**：检查人员尺度不一，导致同一类型问题在不同项目评分差异较大
2. **等级降级执行不到位**：碰触降级情形后未按规定降级，导致等级虚高
3. **整改落实不彻底**：项目部整改回复与实际整改情况不符；整改超期
4. **C/D类项目后续跟踪缺失**：处置后未对项目整改情况进行持续跟踪
5. **客户满意度评价失真**：未真实反映客户评价，或客户投诉未及时处理导致升级
6. **负面警示培训流于形式**：培训考核不严格，人员能力未实质提升
7. **观摩工地创建与评价脱节**：获得观摩称号后现场管理下滑
8. **重大安全/质量隐患瞒报**：现场存在停工/罚款情形但未在评价中体现

### 关键指标

| 指标名称 | 计算方式 | 达标标准 |
|------|------|------|
| 履约品质综合评价平均分 | 所有项目综合评价得分之和/项目总数 | ≥85分（越好） |
| A类项目占比 | A类项目数/在施项目总数×100% | ≤20%（A类有强制分布要求） |
| D类项目发生率 | D类项目数/在施项目总数×100% | 0%或越低越好 |
| 等级降级执行合规率 | 存在降级情形且已降级的项目数/存在降级情形的项目总数×100% | 100% |
| 检查整改按时完成率 | 按期完成整改的项目次数/整改总次数×100% | 100% |
| 客户满意度 | 客户满意度调查得分（满分100） | ≥85分 |
| 安全重大隐患项目发生率 | 发生停工/罚款的政府或客户检查次数/检查总次数×100% | 0% |
| 质量重大隐患项目发生率 | 发生停工/罚款的政府或客户检查次数/检查总次数×100% | 0% |
| 观摩工地创建完成率 | 实际创建观摩工地数/年度目标数×100% | 100% |
| 履约品质提升培训合格率 | 培训合格人数/参训总人数×100% | 100% |
| 履约品质负面舆情发生次数 | 发生次数（越少越好） | 0次 |
