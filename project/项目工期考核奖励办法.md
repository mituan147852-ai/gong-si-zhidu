## 核心条款（要点充沛，引用原文关键表述）

### 第X条（目的）

> "做好工期管理，进一步调动项目部工期管控的积极性，提升项目节点实现率，加速产值释放，同时盘活人力资源，提高资金回收率，特制定本办法。"

### 第X条（适用范围）

> "本办法适用于公司国内工程项目，包括中建一局集团鲁兴达（青岛）建设有限公司、中建一局集团深圳建设有限公司所辖项目，海外工程的工期考核工作由国际工程事业部参考执行。"

### 第X条（管理原则）

> "坚持'公开、公平、公正'的原则，及时有效地完成每项考核；项目工期奖励额度与分公司（事业部）季度履约品质考核结果、人均产值和资金回收率挂钩；项目工期考核以经审批的总进度计划为基准；以全周期现金流预算为基础，分析考核时点项目现金流偏差。"

## 四维宣贯表

### 做什么（要点充沛，8-15条）

1. **工期考核范围确定**：工期考核应覆盖全部在施项目；开工初期、停工状态、完工待验等不具备考核条件的项目需书面说明原因经批准后可不参与考核。
2. **工期考核每半年开展一次**：每年7月和次年1月完成工期考核。
3. **标准奖励额度计算**：标准奖励额度=半年产值（元）×3‰，其中半年产值为监理确认实际产值。
4. **形象进度评价**：以"总进度计划"（须是经甲方签字盖章的有效版本）为依据，核对半年内应完成任务项的实际完成时间偏差和应开始任务项的实际开始时间偏差。
5. **履约品质评价**：分公司/事业部依据《项目履约品质综合评价管理办法》开展所辖项目季度考核，两个季度检查综合得分的平均值作为工期考核履约品质评价系数（L）依据。
6. **工期签证评价**：以项目办理工期签证及时性和是否获得经济补偿确认文件为依据，工期签证以"工程延期报审表"或"主合同补充协议"形式签订。
7. **人均产值统计**：统计考核周期内两个季度人均产值的平均值，确定人均产值系数。
8. **资金回收率核算**：资金回收率=实际回收金额/应回收金额，确定资金回收系数。
9. **工期考核奖励分配**：项目部获得实际奖励额度的90%；分公司/事业部计提以10%为基数，根据生产系统业务能力考核得分确定。
10. **考核结果报送与审核**：分公司/事业部完成考核后报送项目管理部，项目管理部于15个工作日内完成审核及复查。
11. **工期奖励发放决策**：项目管理部汇总考核结果编制工期奖励结果，提交决策会议。
12. **申诉处理**：项目部对考核结果有异议可向公司项目管理部提请复核。
13. **失真处罚**：若发现分公司/事业部考核结果严重失真，每发现一次对有关人员处以10000元以上处罚并限期整改。

### 怎么做（操作流程详细展开，10-20步）

**步骤1：确定考核项目范围**
- 发起部门：分公司/事业部
- 操作动作：梳理所辖全部在施项目名单，确认参与考核项目范围
- 对于不具备考核条件的项目（开工初期、停工状态、完工待验），项目部书面说明原因，分公司/事业部审核后批准可不参与考核
- 输出文件：考核项目范围清单、不具备考核条件项目书面说明
- 签字审批：分公司/事业部总经理

**步骤2：形象进度数据收集**
- 发起部门：项目部
- 操作动作：整理半年内应完成任务项和应开始任务项清单，对比总进度计划（须经甲方签字盖章的有效版本），记录实际完成/开始时间偏差
- 输出文件：形象进度评价数据表、监理确认的实际完成产值文件
- 签字审批：项目经理

**步骤3：履约品质评价**
- 发起部门：分公司/事业部
- 操作动作：依据《项目履约品质综合评价管理办法》开展所辖项目季度履约品质考核，完成两个季度检查
- 根据综合得分平均值和排名确定项目履约品质类别（一类/二类/三类/四类）及对应评价系数（L）
- 输出文件：履约品质检查报告（两个季度）、履约品质评价系数确定表
- 签字审批：分公司/事业部总经理

**步骤4：工期签证办理情况核查**
- 发起部门：项目部
- 操作动作：核查考核周期内造成工期延误的项目，工期签证办理情况
- G=1：已获取工期签证+获得甲方经济补偿确认文件
- G=0.7：已获取工期签证，未获得经济补偿确认文件
- G=0.3：未获取工期签证，但已向甲方发出工期签证函件
- G=0：未获取工期签证且未发出函件
- 输出文件：工期签证文件清单（含"工程延期报审表"或"主合同补充协议"）
- 签字审批：项目经理

**步骤5：人均产值统计**
- 发起部门：项目部
- 操作动作：统计考核周期内两个季度实际完成产值、全口径平均人数，计算季度人均产值
- 公式：季度人均产值=当季度实际完成产值/当季度全口径平均人数
- 公式：人均产值平均值=（Q1人均产值+Q2人均产值）/2
- 人均产值系数查表确定（≥100/87.5/75万/人对应1级/2级/3级项目）
- 输出文件：人均产值统计表
- 签字审批：项目经理

**步骤6：资金回收率核算**
- 发起部门：项目部
- 操作动作：依据合同条款确定应回收金额，统计实际回收金额，计算资金回收率
- 公式：资金回收率=实际回收金额/应回收金额
- 资金回收系数：≥100%对应1；80%-100%按比例；<80%对应0.5
- 输出文件：资金回收率统计表
- 签字审批：项目经理

**步骤7：工期考核综合评价系数计算**
- 发起部门：分公司/事业部
- 操作动作：
  - 形象进度评价系数X = F×0.7 + S×0.3（F=应完成任务项平均得分，S=应开始任务项平均得分）
  - 得分标准：滞后天数t≤0得1分；0＜t≤7得0.6/0.7分；7＜t≤15得0.4/0.5分；15＜t≤30得0.2/0.3分；t＞30得0分
  - 综合评价系数 = X×0.5 + L×0.4 + G×0.1
- 输出文件：工期考核综合评价系数计算表

**步骤8：实际奖励额度计算**
- 发起部门：分公司/事业部
- 操作动作：
  - 标准奖励额度 = 半年产值（元）×3‰
  - 实际奖励额度 = 标准奖励额度×工期考核综合评价系数×人均产值系数×资金回收系数
- 输出文件：实际奖励额度计算表
- 签字审批：分公司/事业部总经理

**步骤9：考核结果报送**
- 发起部门：分公司/事业部
- 操作动作：按附件2《分公司（事业部）工期考核结果模板》格式编制考核结果，附过程考核资料（总进度计划含审批页、监理确认产值文件、工期签证文件、现场考核照片）
- 输出文件：《分公司（事业部）工期考核结果模板》及附件资料
- 签字审批：分公司/事业部总经理
- 报送至：项目管理部

**步骤10：考核结果审核与复查**
- 发起部门：项目管理部
- 操作动作：牵头相关部门审核分公司/事业部报送的考核结果，必要时进行现场复查（复查比例不少于年度考核总数的10%）；如复查结果与考核结果不一致则以复查结果为准
- 输出文件：审核意见、复查报告（如有）
- 签字审批：项目管理部主任
- 时效要求：15个工作日内完成审核及复查

**步骤11：奖励分配比例确定**
- 发起部门：分公司/事业部
- 操作动作：
  - 项目部分配比例：实际奖励额度的90%
  - 分公司/事业部计提比例=10%×分公司/事业部生产系统业务能力考核得分率
  - 班子成员分配比例原则上不大于30%，且不低于整体平均奖金
- 输出文件：奖金分配方案
- 签字审批：分公司/事业部总经理

**步骤12：奖励发放决策**
- 发起部门：项目管理部
- 操作动作：汇总工期考核最终结果，编制工期奖励结果（按附件3《公司总部工期考核报告模板》格式），提交决策会议
- 注意事项：考核时点存贷差超过第一版全周期现金流预算存贷差垫资峰值或垫资期限超过第一版全周期现金流预算的项目，以该两项为否决项，奖金不予发放，考核时点存贷差应计入当期考核拟发放奖励金额
- 输出文件：《公司总部工期考核报告模板》
- 签字审批：决策会议（总经理办公会）

**步骤13：奖励发放执行**
- 发起部门：人力资源部
- 操作动作：根据决策会议结果执行奖金发放
- 注意事项：当期考核奖励额度上限=600万×项目月均人数×3‰，超出部分转入下一考核周期
- 输出文件：奖金发放记录
- 签字审批：人力资源部主任

**步骤14：申诉处理（如有）**
- 发起部门：项目部
- 操作动作：对考核结果有异议，向公司项目管理部提请复核
- 输出文件：申诉书
- 回应部门：项目管理部（受理并复核）

## 核实要点清单（拿着去项目的检查项）

### 资料清单

1. 经甲方签字盖章的《总进度计划》（有效版本）- 核查是否为有效版本
2. 半年内监理确认的实际完成产值文件
3. 形象进度偏差数据表（含应完成任务项、应开始任务项的偏差记录）
4. 工期签证文件（"工程延期报审表"或"主合同补充协议"）- 原件核查
5. 工期签证函件发送记录（如有）- 函件原件+邮寄凭证
6. 甲方经济补偿确认文件（如有）
7. 两个季度履约品质检查报告
8. 履约品质评价系数确定表
9. 人均产值统计表（含实际完成产值、全口径平均人数计算过程）
10. 资金回收率统计表（含应回收金额、实际回收金额依据）
11. 全周期现金流预算方案（第一版）
12. 考核时点存贷差分析表
13. 《分公司（事业部）工期考核结果模板》全套
14. 现场考核照片资料
15. 决策会议纪要（奖励发放决策）
16. 奖金分配方案及发放记录
17. 项目申诉材料及处理结果（如有）
18. 工期奖励纳入成本的财务凭证（商务管理部）
19. 项目月均人数统计表
20. 工期奖励上限计算表

### 常见问题 / 风险点

1. **总进度计划版本无效**：提供的是未经甲方签字盖章的版本，导致形象进度评价缺乏依据
2. **工期签证缺失或不规范**：非我方原因延误后未及时办理"工程延期报审表"，而是采用其他形式（如工作联系单），导致G值偏低（0.3或0）
3. **监理确认产值文件缺失**：无监理签字盖章的产值确认文件，导致标准奖励额度计算缺乏依据
4. **人均产值数据失真**：分母（全口径平均人数）统计口径不一致，导致人均产值系数偏离实际
5. **资金回收率统计错误**：应回收金额确认依据不充分，导致资金回收系数偏低
6. **履约品质评价结果失真**：未按实际排名进行强制分布，或三四类项目数不满足最低要求
7. **存贷差超限未如实披露**：存在否决项但未在考核报告中如实披露，导致奖金违规发放
8. **奖励分配比例超标**：班子成员分配比例超过30%，或低于整体平均奖金

### 关键指标

| 指标名称 | 计算方式 | 达标标准 |
|------|------|------|
| 工期考核覆盖率 | 参与考核项目数/在施项目总数×100% | 100%（除经批准的不可考核项目） |
| 形象进度评价系数X | F×0.7 + S×0.3（滞后天数越少分值越高） | X≥0.7为良好 |
| 履约品质类别达标率 | 一类+二类项目数/考核项目总数×100% | 一类≤20%、二类≤60%、三类≥20% |
| 工期签证合规率 | 获得G≥0.7的工期签证项目数/发生工期延误项目总数×100% | G=1时为最优（有签证+有补偿） |
| 资金回收率 | 实际回收金额/应回收金额×100% | ≥80%（对应资金回收系数0.5以上） |
| 人均产值达标率 | 人均产值达标（≥确保值）项目数/考核项目总数×100% | 越高越好 |
| 考核结果失真率 | 被复查发现严重失真次数/复查项目数×100% | 0% |
| 奖励分配合规率 | 班子成员分配比例≤30%且≥平均奖金的项目数/获奖项目总数×100% | 100% |
| 工期奖励发放及时率 | 在决策后30日内完成发放的项目数/应发放项目总数×100% | 100% |
