## 核心条款

### 目的和依据

满足公司资质、投标备案、履约管理、战略转型及市场行为检查需要，把控注册单位，提高注册效率。依据包括《住房和城乡建设部电视电话会议精神专项治理行动》、《关于一级建造师执业资格实行电子化申报和审批的通知》、《中国建筑一局（集团）有限公司执业资格管理办法》等。

### 适用范围

适用于公司所属各单位。

### 管理原则

- **统筹管理、标准化管理原则**：建立并定期维护各类数据库（证件分库、总库、电子化证件库、实物库）
- **分类管理、分层级管理原则**：分为一级建造师、二级建造师两层级管理

### 职责分工

| 主体 | 职责 |
|------|------|
| 分公司/事业部建造师负责人 | 初步确认注册单位、确认社保与注册单位一致、提交注册、维护台账及数据库、督促延续注册 |
| 建造师本人 | 提交注册申请、跟踪流程、领取证书交项目管理部备案、办理继续教育 |
| 项目管理部 | 审核注册信息表、企业审批、建立证件电子档案、维护数据库、组织继续教育报名 |
| 市场拓展部 | 确定建造师注册单位、审批转社保申请、维护《项目经理使用情况汇总表》 |
| 安全生产监督管理部 | 组织建造师报考/变更安全B本、按月提供新增安全B本名单 |
| 人力资源部 | 盖章提交局人力资源部审批、发放证书津贴和一次性奖励、办理继续教育报销 |

## 四维宣贯表

### 做什么

1. 建立和维护建造师证件数据库（电子档案+实物库）
2. 一级建造师初始注册、变更注册、增项注册、注销注册、重新注册、延续注册
3. 二级建造师注册管理（同上）
4. 建造师电子证书加注及下载（180天使用时限）
5. 安全B本（安全员考核合格证）报考与变更
6. 建造师证书津贴发放（一次性奖励+月度补贴）
7. 继续教育报名、学习、证书领取及延续注册
8. 《项目经理使用情况汇总表》维护
9. 《X年X月新增需考取/变更安全B本人员表》编制（按月）
10. 人证合一核查（注册单位与社保关系所在单位一致）

### 怎么做

**初始注册（一级建造师）**
1. 【建造师】登录中国建造师网，完成用户注册、实名认证
2. 【建造师】登录建设行业执业资格管理信息系统，填写注册信息
3. 【建造师负责人】初步确认注册单位、确认社保关系一致
4. 【建造师】在系统提交初始注册申请
5. 【项目管理部】审核《建造师注册信息表》
6. 【项目管理部】注册在一局：填写《子企业相关业务办理审批单》→ 人力资源部盖章 → 提交一局人力资源部审批
7. 【项目管理部】注册在二公司：登录中国建造师网进行企业审批
8. 【建造师】加注电子证书（180天内）
9. 【建造师】下载电子证书，手写签名及日期（无签名或笔迹不一致=无效）
10. 【建造师】将证书交项目管理部备案
11. 【人力资源部】发放证书津贴（次月生效）

**继续教育**
1. 【建造师】提交《一级注册建造师继续教育培训报名表》至建造师负责人
2. 【项目管理部】汇总后报局人力资源部，上报中建管理学院
3. 【中建管理学院】开通继续教育账户，短信通知建造师
4. 【建造师】登录全国一级注册建造师继续教育平台，缴费、学习
5. 【建造师】学习完毕下载继续教育证书
6. 【人力资源部】办理继续教育费用报销
7. 【建造师】及时办理延续注册

**二级建造师注册**
1. 【建造师】登录北京市住房和城乡建设委员会门户网站，完成用户注册、实名认证
2. 【建造师负责人】确认注册单位
3. 【项目管理部】导出注册申请表，办理签字盖章
4. 【项目管理部】登录北京市住房和城乡建设委员会网站进行企业审批
5. 【建造师】下载打印电子注册证书，交项目管理部备案

### 什么时候做

| 时间节点 | 操作内容 | 执行主体 | 成果文件 |
|----------|---------|---------|---------|
| 实时 | 建造师提交注册申请 | 建造师 | 注册申请表 |
| 每月 | 更新《新增需考取/变更安全B本人员表》 | 安全生产监督管理部 | 月度报表 |
| 每月 | 更新《新增补助人员表》 | 人力资源部 | 月度报表 |
| 每月 | 更新《项目经理使用情况汇总表》 | 市场拓展部/人力资源部 | 动态台账 |
| 180天内 | 电子证书加注（超期无效） | 建造师 | 加注后证书 |
| 证书领取后5日内 | 证书交项目管理部备案 | 建造师 | 备案回执 |
| 继续教育到期前 | 报名、参加学习、下载证书 | 建造师 | 继续教育证书 |
| 继续教育后 | 办理延续注册 | 建造师/项目管理部 | 延续注册证明 |

### 怎么考核

- **考核主体**：项目管理部、安全生产监督管理部、人力资源部
- **考核方式**：证件有效性核查、注册信息比对、台账检查
- **考核周期**：季度/年度
- **奖惩标准**：
  - **人证分离**：零容忍，发现即整改
  - **证件逾期**：未按期办理延续/加注→ 影响投标和备案
  - **证书津贴**：按制度标准发放（取得注册证书+安全B本后次月执行）
  - **一次性奖励**：符合条件者由人力资源部发放

## 核实要点清单

### 资料清单

1. 建造师注册证书（电子证书+纸质版）
2. 建造师社保缴纳记录（核对注册单位一致性）
3. 安全B本（安全生产考核合格证）
4. 继续教育证书（近五年/三年内）
5. 《建造师注册信息表》
6. 《一级注册建造师继续教育培训报名表》
7. 《X年X月新增需考取、变更安全B本人员表》
8. 《X年X月新增补助人员表》
9. 《项目经理使用情况汇总表》
10. 建造师证件台账及数据库（电子档案）
11. 建造师注册系统截图（企业入口核查）
12. 证书津贴发放明细
13. 继续教育发票及报销记录
14. 人证合一核查报告

### 常见问题/风险点

1. **人证分离**：建造师注册单位与社保缴纳单位不一致 → 核查社保记录与注册信息是否一致，发现即整改
2. **电子证书过期**：超过180天未加注新证书 → 核查证书使用时限，逾期无效
3. **证书未手写签名**：电子证书无建造师本人手写签名 → 核查签名笔迹，无签名或不一致=证书无效
4. **安全B本缺失**：建造师无有效安全B本 → 核查安全B本有效期及变更记录
5. **继续教育未完成**：超过有效期未参加继续教育 → 核查继续教育记录，影响延续注册
6. **证件逾期未还**：建造师证件借阅未按期归还 → 核查借阅台账及归还记录
7. **注册信息与台账不符**：系统注册信息与公司台账不一致 → 核查建造师注册系统与企业台账差异
8. **津贴发放错误**：不符合条件人员领取津贴 → 核查发放标准条件（注册证书+安全B本双证齐全）

### 关键指标

| 指标 | 计算方式 | 达标标准 |
|------|---------|---------|
| 人证合一率 | 人证合一人数/建造师总人数×100% | 100%（零容忍） |
| 电子证书有效率 | 有效证书数/在用证书总数×100% | 100% |
| 继续教育完成率 | 完成继续教育人数/应继续教育人数×100% | 100% |
| 安全B本持证率 | 有效B本数/建造师总人数×100% | 100% |
| 证书及时备案率 | 及时备案数/领取证书数×100% | 100% |
| 注册信息准确率 | 信息准确注册数/总注册数×100% | 100% |
