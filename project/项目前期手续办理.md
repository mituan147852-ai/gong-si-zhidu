## 四维宣贯表

### 做什么

- 规范项目开工前期手续办理流程
- 明确各职能部门协同职责
- 确保项目合法合规开工
- 加快手续办理效率

### 怎么做

1. **确认承接方式**：判断是公司名义还是一局名义承接
2. **梳理手续清单**：
   - 印章刻制（项目部章/财务章等）
   - 项目经理授权
   - 介绍信办理
   - 营业执照/资质证书借用
3. **分工协作**：
   - 项目部 → 准备基础材料
   - 法律事务部 → 项目经理授权
   - 综合办公室 → 印章协调
4. **建立台账**：记录办理进度，责任到人

### 什么时候做

| 时间节点 | 内容 |
|----------|------|
| 项目中标后3日内 | 启动手续办理 |
| 开工前30天 | 印章刻制申请 |
| 开工前15天 | 项目经理授权办理 |
| 开工前7天 | 所有手续齐备 |

### 怎么考核

- **考核主体**：项目管理部
- **考核方式**：
  - 手续完备率
  - 办理时效性
  - 合规性检查
- **奖惩措施**：
  - 延误开工 → 追责
  - 手续合规 → 纳入项目考核加分
