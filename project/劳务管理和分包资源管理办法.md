## 核心条款（要点充沛，引用原文关键表述）

### 第X条（目的依据）

> "依据《国务院办公厅关于全面治理拖欠农民工工资问题的意见》（国办发〔2016〕1号）、《保障农民工工资支付条例》（国务院令第724号）、《中国建筑股份有限公司劳务管理手册》（中建股项管字〔2022〕111号）等文件特制定本办法。"

核心目的：全面落实劳务管理方面重要指示及制度文件要求，提升劳务管理水平，推动精细化管理，保障农民工合法权益，维护公司品牌形象。

### 第X条（适用范围）

> "本办法适用于公司各分公司、事业部、项目部以农民工工资支付监督管理为核心的劳务管理工作。"

### 第X条（管理原则）

| 原则 | 说明 |
|------|------|
| 分级管理原则 | 按照公司的组织原则，分公司、事业部逐级对项目劳务和农民工工作进行引领、服务、监督 |
| 全过程管理原则 | 对分包单位农民工工资支付的监督管理从合格分包方准入、招标、合同签订、履约过程、评价考核、不良行为处置形成完整的闭环式管理 |
| 持续改进原则 | 持续改进劳务管理行为，提高管理能力和水平，优化劳务队伍的构成和素质 |
| 以人为本原则 | 各单位应关爱农民工，并为农民工创造良好的工作环境和生活环境 |

### 第X条（职责分工）

| 层级/部门 | 职责 |
|------|------|
| 公司总部项目管理部 | 劳务管理及分包资源管理牵头部门；建立并完善公司劳务管理体系及分包资源管理体系；建立公司分包资源库；建立健全隐患排查、应急管理、责任追究体系；协同财务资金部建立农民工工资支付专用账户、农民工工资保证金管理体系 |
| 分公司/事业部 | 设置专职劳资专管员；建立并完善本单位劳务管理及分包资源管理体系；建立本单位分包资源库；组织管辖项目劳务隐患排查、处理劳务纠纷 |
| 项目部 | 项目经理作为第一责任人；制定劳务管理方案、农民工工资支付方案、应急预案；设立专职劳资专管员并持证上岗；落实属地农民工工资支付专用账户、保证金政策 |
| 分包单位 | 对所招用农民工的实名制管理和工资支付负直接责任；必须配置专职劳资专管员 |

## 四维宣贯表

### 做什么（要点充沛，8-15条）

1. **分包准入管理**：分包商需提交企业相关资料（含资质证书、执业资格证书、社保证明等），经考察并填报准入资料，逐级上报审批后纳入公司合格名录，方可在云筑集采平台进行准入。
2. **入场管理**：严格执行"先签合同后进场"原则；特殊紧急情况需走"应急通道"审批；使用商务管理部发布的最新标准合同版本（含人工费剥离、优先足额支付农民工工资、总包代发等条款）。
3. **进场教育**：对农民工进行权益保护、投诉渠道、遵纪守法等教育培训，考试合格才能进场。
4. **劳动合同管理**：分包单位进场前与农民工签订劳动合同，做到"四明确"（基本信息、计酬方式、支付方式、支付日期）。
5. **实名制管理**：使用建筑工人实名制管理平台（云筑劳务），采用"人脸识别"技术；建立"六统一"台账（身份证、花名册、劳动合同、考勤表、工资表、岗位技能证书）。
6. **维权信息告示牌**：在施工现场醒目位置设立，明示建设单位、总承包单位、最低工资标准、投诉举报电话等基本信息。
7. **工资发放监督管理**：每月收集、检查、核对分包单位农民工花名册、考勤表、工资发放表、银行流水单；进行每期公示。
8. **农民工工资保证金及专用账户**：项目开工前了解属地保证金政策；开设农民工工资专用账户；通过专用账户足额代发（房建项目代发金额不低于总产值的15%）。
9. **退场管理**：监督分包单位结清退场农民工工资；收集退场承诺书或后续支付方式证明；在实名制管理平台办理人员退场。
10. **考核评价管理**：劳务分包每季度考核一次，专业分包每半年考核一次；分优质（≥90分）、合格（60-89分）、不合格（<60分）三级。
11. **投诉及纠纷处理**：以文件、函件、会议纪要、照片、影音等方式留存监督管理痕迹，实现可追溯。
12. **劳务区域联动管理**：一局做出整体划分，联动区域职责单位牵头与属地政府主管部门对接。
13. **农民工群体性事件分级**：按一般（20人以下到公司/局投诉）、较大、重大、特别重大四个级别分级响应。
14. **应急维稳工作**：公司/分公司/项目部三级成立应急领导小组；京内1小时内、京外12小时内到场处理。
15. **监督考核**：公司每半年对分公司/事业部考核；每半年对项目部履约品质考核；每年进行四次劳务隐患排查。

### 怎么做（操作流程详细展开，10-20步）

#### 一、分包准入流程

**步骤1：提交准入资料**
- 发起部门：分包商
- 操作动作：向项目部提交企业相关资料（资质证书、拟派管理人员执业资格证书+社保证明、市场行为合规证明等）
- 输出文件：分包商准入资料包

**步骤2：实地考察**
- 发起部门：项目管理部/分公司/事业部/项目部
- 操作动作：对分包商的生产履约、安全文明施工、工程实体质量、劳动力配置、施工特点特长、综合实力等方面进行实地考察
- 输出文件：资源准入资料及考察意见表

**步骤3：纳入合格名录**
- 发起部门：分公司/事业部
- 操作动作：分公司/事业部单独建立分包合格名录，每季度上报项目管理部汇总
- 劳务队伍：由项目管理部主任和公司主管履约副总经理通过审批后纳入公司合格名录
- 专业分包：由项目经理审批后纳入公司合格名录
- 输出文件：合格分包名录

**步骤4：云筑集采平台准入**
- 发起部门：项目部
- 操作动作：在云筑集采平台中进行准入，填报完整信息，上传有项目经理签字的考察报告扫描件
- 注意：通过其他单位在云筑集采平台中录入的分包商，项目在加入本级操作时仍需上传本项目考察报告
- 输出文件：云筑集采平台准入审批记录

#### 二、分包入场流程

**步骤1：先签合同**
- 发起部门：项目部
- 操作动作：严格执行"先签合同后进场"原则；使用商务管理部发布的最新标准合同版本
- 合同必须包含条款：人工费剥离、优先足额支付农民工工资、总包代发

**步骤2：应急通道（如有紧急情况）**
- 发起部门：项目总监
- 操作动作：提交具体情况书面报告（说明原因+与劳务队伍签订应急范围内的施工协议，明确价格及承包范围）
- 报告需由项目全部责任担当体签字
- 签字审批：上报项目管理部+商务管理部，同时上报公司履约副总和总经济师
- 输出文件：应急通道审批报告

**步骤3：进场教育**
- 发起部门：项目部
- 操作动作：对农民工进行教育培训+考试，考试合格才能进场
- 教育内容：欠薪投诉渠道和维权渠道、农民工权益保护基本知识（分包单位是劳动合同签订主体/工资支付责任主体/清偿责任主体，总承包单位是监督管理单位）、严重违反现场纪律的后果
- 输出文件：进场教育记录、考试试卷

**步骤4：设立维权信息告示牌**
- 发起部门：项目部
- 操作动作：在施工现场醒目位置设立维权信息告示牌
- 内容：建设单位、总承包单位及项目部基本信息；最低工资标准、工资支付日期；劳动保障监察投诉举报电话、劳动争议调解仲裁申请渠道、法律援助申请渠道、公共法律服务热线
- 输出文件：告示牌现场照片

**步骤5：劳动合同签订核查**
- 发起部门：项目部劳资专管员
- 操作动作：监督分包单位与农民工签订劳动合同，检查合法性、合规性、真实性；"四明确"核查（姓名/身份证号/工种/工作期限+计酬方式+支付方式+支付日期）
- 发现问题书面要求分包单位限期整改
- 输出文件：劳动合同核查记录、整改通知单

**步骤6：实名制录入**
- 发起部门：项目部
- 操作动作：将农民工基本信息录入云筑劳务平台（身份信息、签名手印、联系方式、花名册、进退场情况、工作岗位、劳动合同、考勤、工资支付）
- 门禁必须采用"人脸识别"技术（禁止IC/ID卡）
- 输出文件：实名制录入记录

#### 三、工资发放监督管理流程

**步骤1：每月工资表收集**
- 发起部门：分包单位
- 操作动作：分包单位编制工资支付表（农民工本人签字+手印+分包企业公章确认）
- 提交资料：农民工花名册、考勤表、工资发放表、银行流水单

**步骤2：项目部核查**
- 发起部门：项目部劳资专管员
- 操作动作：每月收集、检查、核对上月农民工花名册、考勤表、工资发放表、银行流水单
- 核查"六统一"：身份证、劳动合同、花名册、考勤表、工资发放表（银行流水单）、工资公示表对应统一
- 输出文件：核查记录

**步骤3：公示**
- 发起部门：项目部
- 操作动作：将考勤表、工资发放情况进行每期公示，公示位置应醒目
- 输出文件：公示照片、公示记录

**步骤4：总包代发**
- 发起部门：项目部
- 操作动作：通过农民工工资专用账户直接将工资支付到农民工本人银行账户
- 房建项目代发金额不低于总产值的15%
- 分包单位应向项目部出具当期农民工工资已全部支付、不存在拖欠的证明
- 输出文件：代发凭证、不拖欠证明

**步骤5：处理投诉或问题**
- 发起部门：项目部
- 操作动作：收到投诉→尽快核实→属实则要求分包单位限期整改；不实则明确告知投诉人；如发生分包款纠纷→识别劳务费部分保证不拖欠
- 书面通知分包单位并按分包合同约定做书面告诫或要求限期整改或收取违约金
- 拒不整改→向公司总部申请暂停投标资格、清出合格分供方名录、录入不良行为名录
- 输出文件：投诉处理记录、整改通知单

#### 四、考核评价流程

**步骤1：季度/半年考核**
- 发起部门：项目经理
- 操作动作：项目经理牵头对考核期内所有参与施工的分包单位进行考核评分
- 劳务分包：每季度一次；专业分包：每半年一次
- 评分内容：整体素质、资质资信、施工能力、履约能力、劳动力配置、工程质量、文明施工、安全生产、实名制管理、总包代发工资、与项目部配合度、遵纪守法、结算溢价、恶性上访讨薪事件
- 输出文件：分包考核评分表

**步骤2：考核结果定级**
- 发起部门：项目部
- 操作动作：
  - 优质分包：≥90分→鼓励使用
  - 合格分包：60-89分→酌情使用
  - 不合格分包：<60分→严禁使用
  - 发生恶性群体性事件→直接列入不合格
  - 维保期间不配合→不可评为A类（优质）
- 输出文件：考核结果汇总表

**步骤3：结果应用**
- 发起部门：项目部/分公司/事业部
- 操作动作：将考核结果录入分包管理档案；对不合格分包启动淘汰禁用程序；优质分包优先推荐使用
- 不良行为分包处置：一般不良行为禁入期3年；较重不良行为禁入期5年；严重不良行为永久禁入
- 输出文件：不良行为处分报告

#### 五、群体性事件处置流程

**步骤1：事件发生**
- 发起部门：项目部
- 操作动作：农民工到公司/局/集团/政府部门上访讨薪，项目部立即启动应急预案
- 京内项目：项目经理及相关人员1小时内到达事发地点
- 京外项目：12小时内到达事发地点

**步骤2：到场处置**
- 发起部门：项目经理（或责任担当体）
- 操作动作：第一时间到现场处理，控制事态发展，落实纠纷原因，制定解决方案
- 如项目经理有极特殊原因不能按时到场→向公司主管生产履约副总请示，说明原因并提出解决方案，安排其他责任担当体按时到场
- 输出文件：事件处置记录

**步骤3：事件分级与上报**
- 发起部门：项目部
- 操作动作：按四个级别（一般/较大/重大/特别重大）确定事件级别，向对应层级上报
- 属地涉事项目/分公司/事业部相关人员1小时内到达现场处置
- 输出文件：事件分级报告

**步骤4：责任追究**
- 发起部门：公司总部
- 操作动作：根据造成的社会影响、政府部门及上级单位的处理意见、事件性质及管理问题，依据《中国建筑股份有限公司分包单位农民工工资支付监督管理办法》《中国建筑一局（集团）有限公司劳务管理办法》《中建一局集团第二建筑有限公司底线管理办法》进行处理及责任追究
- 输出文件：责任追究报告

### 什么时候做（时间节点具体）

| 时间节点 | 操作内容 | 执行主体 | 成果文件 |
|------|------|------|------|
| 进场前 | 完成分包合同签订 | 项目部+分包单位 | 分包合同 |
| 进场前 | 完成进场教育并考试 | 项目部 | 教育记录+考试试卷 |
| 进场前 | 农民工签订劳动合同（四明确） | 分包单位+项目部核查 | 劳动合同 |
| 进场3日内 | 完成实名制录入（云筑劳务） | 项目部 | 实名制录入记录 |
| 进场5日内 | 设立维权信息告示牌 | 项目部 | 告示牌+现场照片 |
| 每月 | 收集核查分包单位花名册、考勤表、工资表、银行流水单 | 项目部劳资专管员 | 核查记录 |
| 每月 | 考勤和工资情况公示 | 项目部 | 公示记录+照片 |
| 每月 | 通过专用账户代发工资 | 项目部 | 代发凭证 |
| 每月 | 分包单位出具不拖欠证明 | 分包单位 | 不拖欠证明 |
| 农民工退场时 | 结清工资并收集退场承诺书/后续支付证明 | 分包单位+项目部 | 退场承诺书 |
| 农民工退场后 | 在实名制管理平台办理人员退场 | 项目部 | 退场记录 |
| 劳务分包每季度 | 对分包单位进行考核评价 | 项目经理 | 分包考核评分表 |
| 专业分包每半年 | 对分包单位进行考核评价 | 项目经理 | 分包考核评分表 |
| 每年四次 | 劳务隐患排查（真实准确反馈） | 分公司/事业部、项目部 | 排查报告 |
| 公司每半年 | 对分公司/事业部进行考核 | 公司项目管理部 | 考核报告 |
| 公司每半年 | 对项目部进行履约品质考核（重点：体系建设、方案预案、实名制、代发） | 公司项目管理部 | 考核报告 |
| 事件发生时 | 京内1小时内/京外12小时内到场处置 | 项目经理/责任担当体 | 处置记录 |
| 事件发生后2小时内 | 向公司总部汇报 | 项目部 | 上报记录 |

### 怎么考核（考核标准明确）

| 考核主体 | 考核方式 | 考核周期 | 奖惩标准 |
|------|------|------|------|
| 公司项目管理部 | 对分公司/事业部劳务管理重点指标完成情况考核 | 半年 | 依据底线管理办法执行 |
| 公司项目管理部 | 对项目部履约品质考核（重点：体系建设、方案预案、实名制、代发） | 半年 | 现场随机抽取劳务人员印证实名制资料真实情况，发现一人缺失或严重不符→实名制管理为零分 |
| 项目经理 | 对分包单位考核评价（劳务季度、专业半年） | 季度/半年 | 90分及以上→优质（鼓励使用）；60-89分→合格（酌情使用）；<60分→不合格（严禁使用） |
| 公司项目管理部 | 劳务隐患排查 | 每年四次 | 谎报、瞒报、不报→依规处理 |

**不良分包处置标准**：

| 行为类型 | 处置方式 |
|------|------|
| 一般不良行为 | 禁入期3年 |
| 较重不良行为 | 禁入期5年 |
| 严重不良行为 | 永久禁入 |
| 恶性群体性事件 | 直接列入不合格，在公司范围内严禁使用 |
| 维保期间不配合 | 不可评为优质分包 |

### 常见问题 / 风险点

1. **先签合同后进场执行不到位**：分包队伍先进场施工，后签合同
2. **劳动合同不规范**：劳动合同缺少"四明确"中任一项；非工人本人签字；无分包企业公章
3. **实名制管理不实**：未使用云筑劳务平台；门禁未采用人脸识别（仍在用IC/ID卡）；花名册与实际人员不符
4. **工资代发不合规**：代发金额低于总产值的15%；未通过专用账户发放；分包单位扣押农民工社保卡
5. **"六统一"不对应**：身份证、劳动合同、花名册、考勤表、工资表、银行流水单存在不对应项
6. **考勤表、工资表未公示或公示位置不醒目**
7. **分包考核评价不客观**：评分随意，未按真实情况评分
8. **农民工退场时工资未结清**：退场承诺书缺失；分期支付协议未签订
