## 核心条款（要点充沛，引用原文关键表述）

### 第X条（目的）

> "坚持节约资源和保护环境的基本国策，全面推进'五位一体'总体布局，践行'绿水青山就是金山银山'的理念，严格履行环境保护主体责任，促进公司生产经营过程中资源的节约利用和生态环境保护工作，着力推动绿色发展和高质量发展，履行企业社会责任和使命。"

### 第X条（适用范围）

> "适用于公司的环境管理，管理范围覆盖公司各办公领域和房建、市政、路桥、装饰装修、机电安装、土石方工程等各施工领域。"

### 第X条（管理原则）

> "全面落实公司环境管理主体责任，坚持'党政同责，一岗双责'；全员参与，明确各岗位的环境管理职责、监督职责，做到监管分离、监管并重，遵循持续改进的原则；严格执行相关法律法规，将法律法规要求作为企业环境管理坚守的底线。"

### 第X条（职责分工）

| 层级/部门 | 职责 |
|------|------|
| 公司项目管理部 | 环境管理主管部门，负责对公司环境问题的管理与指导；对环境管理体系运行控制情况进行监督；对各单位环境目标、指标及方案的完成情况进行检查；组织对检查中不符合项制定纠正措施 |
| 公司商务管理部 | 对全公司各项目的环境管理措施成本投入进行审核 |
| 公司科技质量部 | 建立公司节能环保技术保证体系，开展节能环保技术研究，推广先进技术；组织实施节能环保技术研发工作；审核批准重要节能环保技术方案 |
| 公司办公室/党委办公室/董事会办公室 | 对公司机关所属范围内的大气、水体、固体废弃物进行控制 |
| 分公司/事业部 | 对本部门环境因素进行识别、评价和控制；建立环境管理体系；对所属项目环境目标、指标及方案的完成情况进行检查 |
| 项目部 | 对本单位环境因素进行识别、评价和控制；施工现场重要环境因素控制措施的制定和具体实施；负责大气、水、土壤、噪声、光、固体废弃物的管理进行监督检查 |
| 公司总部各部室 | 对本部门的环境因素进行识别、评价和控制 |

## 四维宣贯表

### 做什么（要点充沛，8-15条）

1. **组织建立**：每年年初建立生态环保责任体系。
2. **制度编制与修订**：每年年初编制并修订总部及分公司/事业部各职能部门和项目层面生态环保制度。
3. **环境因素识别与评价**：公司每年一季度确认环境因素并更新重要环境因素清单；新开工项目进场后一个月内完成环境因素辨识评价；各单位每年一季度对环境因素变化进行确认。
4. **节能环保策划**：施工项目进场后一个月内编制节能环保策划方案（含工程概况、目标、机构职责、重要环境因素清单、控制措施清单、应急措施等）。
5. **环境方案管理**：项目部编制环境方案（扬尘治理方案、建筑垃圾处置方案、污水治理方案等），由项目经理审批并报监理备案。
6. **大气污染控制**：扬尘控制（湿法作业、覆盖、硬化道路、喷淋降尘等）；废气排放控制（设备维护保养、食堂油烟净化等）。
7. **水污染控制**：三级沉淀池设置；隔油池；化粪池；办理污水排放许可手续。
8. **噪声污染控制**：严控作业时间（晚22点至次日6点禁止噪声作业）；委外监测并保留记录。
9. **固体废弃物及危险废物污染控制**：分类收集（建筑垃圾/生活垃圾/有毒有害垃圾）；委托有资质单位处理；建立废弃物处置台账。
10. **节能减排控制措施**：制定节能措施；办公区/施工区/生活区分表计量；节能灯具；统计分析和报表报送。
11. **环境应急管理**：编制《空气重污染应急预案》和《环境问题应急预案》；建立预警机制；发生环境事件立即启动应急预案并上报。
12. **监督与检查**：分公司/事业部每季度检查；公司总部每半年对所有在施项目进行检查；项目部每月不少于一次自查。
13. **激励与约束**：获得国家级/省部级生态环保荣誉称号分别奖励8万/5万元；发生一般及以上突发环境事件按"三重一大"进行处罚；拒不整改对第一责任人处罚1000元。
14. **生态环保责任书签订**：新开项目开工前签订生态环保责任书，落实工作目标及控制指标分解。
15. **培训与宣传**：分公司/事业部开展环保业务培训；开展世界环境日、全国低碳日等活动。

### 怎么做（操作流程详细展开，10-20步）

#### 一、环境因素识别与评价流程

**步骤1：环境因素识别**
- 发起部门：项目部/分公司/事业部/公司总部
- 操作动作：识别办公活动、施工活动、产品实现等全过程的环境因素
- 识别内容：大气污染、水污染、土壤污染、噪声、振动、资源消耗、有毒有害化学品使用、臭氧层破坏、电磁辐射、光热辐射、恶臭等
- 输出文件：《环境因素识别评价表》（附件1）
- 签字审批：填报人

**步骤2：环境因素评价**
- 发起部门：项目部/分公司/事业部
- 操作动作：使用《环境影响评分基准》打分，当a+b+c+d+e+f>14或g+h>7时确定为重要环境因素
- 评价维度：影响范围、影响程度、产生量、发生频次、法规符合性、社区关注程度、资源消耗、可节约程度
- 输出文件：《重要环境因素清单》（附件2）
- 签字审批：填报人→审核人→审批人

**步骤3：清单上报与审批**
- 发起部门：项目部
- 操作动作：新开工项目进场后一个月内完成辨识评价，经项目经理审核后报公司项目管理部审批
- 公司每年一季度在上年基础上确认并更新公司重要环境因素清单
- 输出文件：《重要环境因素清单》

**步骤4：清单更新触发**
- 发起部门：项目部/公司总部
- 操作动作：当发生以下情况时及时组织再识别与评价：法律法规变更；公司活动/产品/服务发生较大变化；相关方有合理报怨
- 输出文件：更新后的《重要环境因素清单》

#### 二、节能环保策划流程

**步骤1：策划编制**
- 发起部门：项目总工程师
- 操作动作：进场后一个月内结合本项目环境因素评价结果编制节能环保策划方案
- 策划内容：工程概况；编制依据；节能环保目标；节能环保机构及职责；环境因素评价表和重要环境因素清单；与评价表对应的控制措施清单及责任清单；重要环境因素措施效果分析和效益目标；节能环保保障（技术/资金/人员/物资设备等投入）；应急措施
- 输出文件：《节能环保策划方案》
- 签字审批：项目总工程师→项目经理

**步骤2：策划审批**
- 发起部门：项目管理部
- 操作动作：收到项目上报的节能环保策划方案后进行审批
- 输出文件：审批意见
- 签字审批：项目管理部主任

**步骤3：策划落实**
- 发起部门：项目部
- 操作动作：在项目实施过程中严格落实策划内容
- 公司总部应对策划的编制和落实情况进行监督
- 输出文件：策划执行记录

#### 三、环境方案管理流程

**步骤1：方案编制**
- 发起部门：项目部
- 操作动作：编制环境方案（扬尘治理方案、建筑垃圾处置方案、污水治理方案等）
- 方案内容：管理目标；控制措施和方法；职能和层次职责和权限；实现目标所需的资源及提供方式
- 输出文件：环境方案文件
- 签字审批：项目经理

**步骤2：方案备案**
- 发起部门：项目部
- 操作动作：将环境方案报监理单位备案
- 输出文件：监理备案回执

**步骤3：方案年度评审**
- 发起部门：项目部
- 操作动作：每年年初对环境方案进行评审，如有以下情况需重新修订：合同条件重大变更；项目组织机构重大调整；应对环境管理方案；工程重大变化原策划结果无法实现；法律法规重大变化
- 修订内容应得到原批准人批准，并通知文本持有者
- 输出文件：修订后的环境方案

#### 四、环境控制实施流程

**步骤1：大气污染控制**
- 发起部门：项目部
- 操作动作：
  - 土方挖运卸填采取湿法作业；重污染预警期间及四级以上大风天气不得进行土方作业
  - 现场裸露场地绿化或覆盖处理
  - 出入口设置洗车设备+三级沉淀池；运输车辆封闭不得超量装载
  - 细散颗粒材料封闭或有覆盖措施
  - 施工现场垃圾站密闭处理
  - 外脚手架/密目网及时清理
  - 机械设备定期维护保养；食堂设置油烟净化装置
- 输出文件：扬尘控制记录

**步骤2：水污染控制**
- 发起部门：项目部
- 操作动作：
  - 洗车池配备三级沉淀池；与市政污水管网相接
  - 食堂设置隔油池；严禁将食物废料倒入下水道
  - 设置化粪池；生活污水经沉淀后排放
  - 办理污水排放许可手续
- 输出文件：水污染控制记录

**步骤3：噪声污染控制**
- 发起部门：项目部
- 操作动作：
  - 严控作业时间：晚22点至次日6点禁止噪声作业；超55dB工序严禁施工；特殊需要须办夜间施工证
  - 钢筋集中加工；混凝土/砂浆集中搅拌；木材集中加工
  - 进行噪声监测（必要时委外）
- 噪声标准：土方白天<75dB夜间<55dB；结构工程白天<70dB夜间<55dB；装修工程白天<65dB夜间<55dB
- 输出文件：噪声监测记录、夜间施工许可证

**步骤4：固体废弃物管理**
- 发起部门：项目部
- 操作动作：
  - 设置废弃物分类箱（建筑垃圾/生活垃圾/有毒有害垃圾）；设置全封闭垃圾站
  - 生活垃圾与无毒无害建筑垃圾：与有消纳资格单位签订委托处理协议
  - 有毒有害建筑垃圾：与供应商/分包方签订处理协议
  - 废弃物投放做到分类清运、不遗洒不混放
- 输出文件：废弃物处置台账、委托处理协议

**步骤5：节能减排控制**
- 发起部门：项目部
- 操作动作：
  - 施工区/办公区/生活区分表计量
  - 使用节能灯具和声、光控开关
  - 合理安排施工区域及顺序；选择功率与负荷匹配的机械设备
  - 每月填报能源和水等节能减排报表
- 输出文件：能源消耗统计表、节能减排报表

#### 五、环境应急管理流程

**步骤1：应急预案编制**
- 发起部门：项目部
- 操作动作：编制《空气重污染应急预案》和《环境问题应急预案》
- 内容：应急组织机构；预警机制；应急响应程序；应急处置措施；信息报送程序
- 输出文件：《应急预案》
- 签字审批：项目经理

**步骤2：预警信息传递**
- 发起部门：分公司/事业部/项目部
- 操作动作：将政府有关部门发布的预警信息及时传递到生产一线
- 根据预警级别（一级/二级/三级/四级）采取限产减产措施并加强污染控制措施

**步骤3：事件发生响应**
- 发起部门：项目部
- 操作动作：发生环境污染事件后立即启动应急预案，开展应急处置工作
- 按应急等级（一级/二级/三级）启动相应预案
- 及时向上级单位/政府主管部门报告
- 输出文件：应急处置记录

**步骤4：舆情巡查**
- 发起部门：项目部/分公司/事业部
- 操作动作：建立舆情巡查及信息报送制度，定期开展网络巡查
- 发现负面舆情/违法失信行为第一时间上报公司项目管理部并立即采取措施
- 输出文件：舆情巡查记录

#### 六、监督与检查流程

**步骤1：项目部自查**
- 发起部门：项目经理
- 操作动作：项目部每月不少于一次由项目经理组织，环境管理员配合，相关部门参与的环境保护工作检查
- 发现问题当场整改；不能整改的向作业单位/部门签发"施工现场安全检查记录"限期整改
- 输出文件：检查记录、整改通知单及回复

**步骤2：分公司/事业部季度检查**
- 发起部门：分公司/事业部
- 操作动作：每季度对项目部的环境保护管理工作进行检查
- 检查内容：环境管理体系、目标指标实施情况、重要环境因素控制措施运行情况、方案落实情况
- 输出文件：检查报告
- 报送至：公司总部项目管理部

**步骤3：公司总部半年检查**
- 发起部门：项目管理部
- 操作动作：公司总部每月对重点区域/环保事件频发区域项目进行抽查；每半年对所有在施项目进行检查
- 发现当场不能整改完善的环境因素签发"检查记录"限期整改，并进行复查
- 输出文件：检查记录、复查记录

**步骤4：环境管理考核**
- 发起部门：项目管理部
- 操作动作：将环境管理纳入公司对项目半年考核的评价体系；纳入分公司/事业部及项目部年度目标考核评价体系
- 输出文件：考核评价报告

### 常见问题 / 风险点

1. **环境因素识别不全面**：只识别了施工阶段的因素，未覆盖办公活动、拆除阶段等全生命周期
2. **节能环保策划与实际脱节**：策划内容笼统，未结合项目具体环境因素制定可操作措施
3. **扬尘控制措施不到位**：重污染预警期间违规进行土方作业；喷淋降尘设施未正常运行
4. **三级沉淀池形同虚设**：沉淀池未及时清理维护，污水未经沉淀直接排放
5. **噪声超标未监测**：未按要求进行噪声监测，或监测频次不够，夜间施工许可证缺失
6. **废弃物分类不到位**：有毒有害废弃物与普通建筑垃圾混放；处置协议不规范
7. **节能减排统计不实**：分表计量形同虚设；能源消耗数据与实际偏差较大
8. **应急预案缺乏针对性**：预案内容与项目实际情况不符；未按要求定期演练
