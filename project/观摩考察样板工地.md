## 核心条款

### 目的和依据

坚持以习近平新时代中国特色社会主义思想为指导，落实局"以客户为中心的品牌强企"战略，聚焦公司"一创三领先"战略愿景，不断提升项目履约管理水平，切实发挥观摩工地的示范引领作用，以优质的现场管理提高公司品牌影响力。依据《中国建筑一局（集团）有限公司观摩考察样板工地管理办法》编制。

### 适用范围

本细则适用于各分公司、事业部及创建观摩考察样板工地的项目。

### 管理原则

- **服务市场原则**：满足局"一化三线"市场营销战略布局，覆盖公司主要经营区域
- **品质优良原则**：保证过程履约品质优良，充分展示项目管理水平
- **资源共享原则**：落实"我为人人、人人为我"的观摩工地创建初衷，成果共享

### 职责分工

| 层级 | 职责 |
|------|------|
| 公司总部 | 修订实施细则、落实局目标、制定年度计划、监督落实、协助承办重大观摩活动、品牌传播 |
| 分公司/事业部 | 落实年度创建指标、组织策划、跟踪落实、指导项目、牵头承办外部重大观摩活动、宣传报道 |
| 项目部 | 作为实施主体、全面落实各项工作、编制实施策划、主动对接外部单位、配合营销观摩 |

## 四维宣贯表

### 做什么

1. 制定年度观摩工地创建计划（年初）
2. 编制观摩工地创建实施策划
3. 按计划落实观摩工地创建各项工作
4. 组织开展预验收（上、下半年各一次）
5. 迎接局正式验收评价
6. 承办外部重大观摩活动（政府/协会/中建集团/大客户）
7. 挖掘总结项目管理亮点并对外宣传报道
8. 配合市场营销观摩考察活动
9. 建立观摩工地台账及成果共享
10. 年度目标完成情况纳入经营业绩考核

### 怎么做

1. 【分公司/事业部】梳理年度代表工程，优选拟创建项目
2. 【分公司/事业部】确认创建计划，上报项目管理部
3. 【项目管理部】汇总确认后上报一局（年度计划原则上不调整）
4. 【项目部】编制观摩工地创建实施策划，明确责任人及工作措施
5. 【项目经理】作为第一责任人，监督策划落实
6. 【分公司/事业部】定期跟踪实施情况，日常指导
7. 【分公司/事业部】上半年/下半年各组织一次预验收
8. 【项目管理部】对照验收标准核查（观摩得分/履约评价得分）
9. 【项目部】承办外部重大观摩活动（政府/协会/大客户）
10. 【项目管理部】组织通过局正式验收（高端/省级/市级）
11. 【项目管理部】未通过局验收的，组织公司级验收
12. 【相关部门】在企业主流宣传平台传播亮点
13. 【项目部】配合营销观摩考察活动，收集观摩信息

### 什么时候做

| 时间节点 | 操作内容 | 执行主体 | 成果文件 |
|----------|---------|---------|---------|
| 年初 | 制定年度观摩工地创建计划 | 分公司/事业部 | 年度创建计划 |
| 全年 | 持续落实观摩工地创建实施策划 | 项目部 | 实施策划 |
| 上半年 | 预验收 | 分公司/事业部 | 预验收记录 |
| 下半年 | 预验收 | 分公司/事业部 | 预验收记录 |
| 随时 | 承办外部重大观摩活动 | 项目部 | 活动记录 |
| 随时 | 配合营销观摩考察 | 项目部 | 观摩信息记录 |
| 年度 | 验收评价 | 局/公司 | 验收评定结果 |
| 年度 | 目标完成情况纳入考核 | 分公司/事业部 | 考核评价 |

### 怎么考核

- **考核主体**：公司项目管理部、分公司/事业部经营业绩考核组
- **考核方式**：目标达成检查、验收评价打分
- **考核周期**：年度
- **奖惩标准**：
  - 通过中建集团观摩验收 → 公司奖励 **50万元**
  - 局评定为高端观摩工地 → 奖励 **30万元**（局+公司累计）
  - 局评定为省级观摩工地 → 公司奖励 **20万元**
  - 局评定为市级观摩工地 → 公司奖励 **10万元**
  - 公司评定为公司级观摩工地 → 公司奖励 **10万元**
  - 不同级别观摩工地 → 按最高标准发放，不重复计算
  - 观摩工地奖励由公司承担，不纳入项目成本

## 核实要点清单

### 资料清单

1. 年度观摩工地创建计划及上报记录
2. 观摩工地创建实施策划（明确责任人/措施/节点）
3. 预验收记录（上/下半年各一次）
4. 观摩工地验收评价表（局下发标准）
5. 项目履约品质综合评价得分（≥95分/90分）
6. 观摩考察样板工地验收得分（高端≥95/省级≥95/市级≥90）
7. 承办重大观摩活动证明（政府批文/协会通知/活动照片）
8. 对外宣传报道记录（企业主流宣传平台）
9. 观摩信息收集记录（观摩单位/人员/反馈）
10. 安全生产标准化建设资料

### 常见问题/风险点

1. **创建目标不完成**：在施项目超过5个但未创建两项观摩工地 → 核查是否提交书面申请说明
2. **预验收流于形式**：未按上/下半年组织预验收 → 核查预验收记录时间节点
3. **验收得分不达标**：观摩得分或履约评价得分低于标准 → 核查两项得分是否达标（高端/省级≥95/市级≥90）
4. **重大观摩活动缺失**：高端/省级观摩工地无相应级别活动承办记录 → 核查活动批文和级别证明
5. **实施策划落实不到位**：策划内容未有效执行 → 对照策划检查实际工作落实情况
6. **宣传报道不到位**：亮点未在企业平台传播 → 核查发布记录
7. **计划随意调整**：年度计划外调整未走报告审批 → 核查调整原因和审批记录

### 关键指标

| 指标 | 计算方式 | 达标标准 |
|------|---------|---------|
| 观摩工地创建数量 | 各单位年度实际创建数量 | 分公司≥1个/年；在施≥5个≥2个/年 |
| 观摩考察得分 | 局验收评价得分 | 高端/省级≥95分；市级≥90分 |
| 履约品质评价得分 | 一局项目履约品质综合评价 | 高端/省级≥95分；市级≥90分 |
| 预验收执行率 | 实际预验收次数/应预验收次数×100% | 100% |
| 观摩活动承办率 | 实际承办重大活动数/计划数×100% | 100% |
| 公司代表性工程观摩覆盖率 | 代表性工程项目有观摩记录/总数×100% | 100% |
