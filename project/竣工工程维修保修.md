# 竣工工程维修保修及保修金回收管理实施细则

## 基本信息

| 项目 | 内容 |
|------|------|
| 制度名称 | 中建一局集团第二建筑有限公司竣工工程维修保修及保修金回收管理实施细则 |

## 核心条款

### 目的和依据

依据《建设工程质量管理条例》、《房屋建筑工程质量保修办法》、《中国建筑一局（集团）有限公司竣工工程保修和保修管理办法》，提升产品全生命周期服务能力，规范工程交付使用后工程维保和保修金回收工作机制，明确工作标准，提高工作质量和效率，保持良好客户关系，提高客户满意度。

### 适用范围

适用于公司所属所有单位。

### 关键术语

- **工程保修期**：从竣工日期起算，按合同约定，最不低于法定最低年限（地基基础/主体结构为设计文件规定合理使用年限；防水5年；供热供冷2个采暖/供冷期；电气管线/给排水/设备安装/装修2年）
- **缺陷责任期**：从工程实际通过竣工验收之日起计算，最长不超过24个月
- **工程保修金**：发包人从应付工程款中预留的质量保证资金，不得超过工程价款结算总额的3%

### 职责分工

| 主体 | 职责 |
|------|------|
| 项目管理部 | 建立三级维保体系（公司/分公司/项目部）、监督维保工作、成本锁定确认、组织保修金回收、督导奖罚 |
| 科技质量部 | 监督指导质量缺陷维保工作、审批重大维修技术方案 |
| 商务管理部 | 确定保修金金额、审核维修方案造价 |
| 财务资金部 | 保修金回收额度确认 |
| 法律事务部 | 指导法律手段解决保修及保修金催收、案件策划审批 |
| 分公司/事业部 | 监管竣工项目维保工作、每季度末收集保修单及回访表上报 |
| 项目部 | 办理交接手续、签订质量保修书、确定维保负责人、提交产品说明书、维修实施、回访、办理缺陷责任期终止手续、回收保修金 |

## 四维宣贯表

### 做什么

1. 建立公司总部/分公司/项目部三级维保管理体系
2. 项目交付后办理交接手续，签订质量保修书（30日内报项目管理部备案）
3. 向客户提交工程产品说明书（含维保客服联系方式）
4. 保修期内提供维修服务（保修销项率100%，工程交付1年内常驻人员）
5. 登记《工程质量问题报修单》，当日通知维保负责人
6. 踏勘核对报修情况，判定保修责任（24小时内到场核查，48小时内维修）
7. 10万元以上维修编制专项方案（科技质量部审批）
8. 配合客户进行试验试运行并办理确认手续
9. 维修完毕24小时内通知客户验收，书面确认
10. 每次维修后对客户进行回访（《维修情况及客户意见表》，按季度上报）
11. 竣工后进行成本锁定前提交保修费用预留申请（上限不超过质保金额10%）
12. 完成竣工结算后15个工作日内确认保修金并报送详细信息
13. 缺陷责任期届满，办理缺陷责任期终止手续
14. 保修金回收的具体实施
15. 分包合同中约定保修责任，同比例预留分包质量保证金

### 怎么做

1. 【项目部】与客户办理交接手续，签订质量保修书
2. 【项目部】30日内将质量保修书上报公司总部项目管理部备案
3. 【项目部】向客户提交工程产品说明书（含维保人员联系方式）
4. 【项目部】工程交付后1年内安排常驻保修人员（保修销项率100%）
5. 【项目部/保修人员】收到保修通知后，当日记录《工程质量问题报修单》
6. 【项目部/保修人员】48小时内到场核查，确认保修范围
7. 【属于保修范围】立即组织维修；【不属于保修范围】向客户说明，协助客户维修
8. 【10万元以上维修】编制专项维修方案，提交科技质量部审批
9. 【涉及结构安全】由原设计或相应资质设计单位出具维修方案
10. 【维修实施】根据方案制定预算，报商务管理部审核
11. 【维修完成】24小时内通知客户验收，办理书面确认
12. 【项目部/分公司】每次维修后进行客户回访，填写《维修情况及客户意见表》
13. 【分公司/事业部】每季度末汇总保修单及回访表，上报项目管理部
14. 【项目部】工程竣工后、成本锁定前，提交保修费用预留申请（上限10%）
15. 【项目部】完成竣工结算后15个工作日内，上报保修金详细信息
16. 【缺陷责任期满】与客户办理缺陷责任期终止手续
17. 【项目部】启动保修金回收，收回保修金

### 什么时候做

| 时间节点 | 操作内容 | 执行主体 | 成果文件 |
|----------|---------|---------|---------|
| 工程交付时 | 办理交接手续，签订质量保修书 | 项目部 | 质量保修书 |
| 交付后30日内 | 质量保修书报项目管理部备案 | 项目部 | 备案回执 |
| 工程交付后1年内 | 常驻保修人员到位 | 项目部 | 人员名单 |
| 接到保修通知 | 当日记录保修单 | 保修人员 | 工程质量问题报修单 |
| 接到保修通知48小时内 | 到场核查并维修 | 保修人员 | 维修记录 |
| 10万元以上维修 | 编制方案报科技质量部审批 | 项目部/科技质量部 | 维修方案审批单 |
| 维修完成24小时内 | 通知客户验收确认 | 项目部 | 验收确认单 |
| 每次维修后 | 客户回访并填表 | 项目部/分公司 | 维修情况及客户意见表 |
| 每季度末 | 汇总保修单及回访表上报 | 分公司/事业部 | 季度汇总表 |
| 工程竣工后/成本锁定前 | 保修费用预留申请 | 项目部 | 预留申请单 |
| 竣工结算后15个工作日内 | 确认保修金信息并上报 | 项目部 | 保修金详细信息表 |
| 缺陷责任期满 | 办理缺陷责任期终止手续 | 项目部 | 终止确认书 |
| 缺陷责任期满后 | 实施保修金回收 | 项目部 | 回收凭证 |

### 怎么考核

- **考核主体**：项目管理部（公司总部）、商务管理部、财务资金部、法律事务部
- **考核方式**：保修金台账跟踪、专项检查、维修回访记录核查
- **考核周期**：季度汇总、年度考核
- **奖惩标准**：
  - **保修金回收奖罚**：按回收难易度和回收比例，给予责任团队奖励；回收不力扣减绩效
  - **保修费用超限**：超过质保金10%预留上限需专项报告审批
  - **维修不及时**：客户投诉或造成损失追究责任
  - **分包保修金扣留**：分包结算款中同比例预留，未扣留造成的损失由项目部承担

## 核实要点清单

### 资料清单

1. 工程质量保修书（交付时签订）
2. 工程产品说明书（含维保联系人）
3. 工程质量问题报修单（每次保修均有）
4. 维修情况及客户意见表（每次维修后）
5. 维修方案及科技质量部审批记录（10万元以上）
6. 维修验收确认单（每次维修后）
7. 客户回访记录（按季度）
8. 保修费用预留申请及审批
9. 竣工结算资料
10. 保修金详细信息台账（项目名称/竣工时间/缺陷责任期/结算金额/保修金比例/回收责任人）
11. 分包合同中保修责任约定及保证金扣留记录
12. 缺陷责任期终止手续
13. 保修金回收凭证
14. 成本锁定中维保费用确认资料
15. 法律诉讼案件相关资料（如有）

### 常见问题/风险点

1. **保修响应不及时**：超过48小时到场或维修 → 核查报修单时间与维修记录时间间隔
2. **保修责任判定错误**：将非保修范围列为保修范围或反之 → 核查保修单与合同保修范围是否一致
3. **维修记录缺失**：维修后无书面确认 → 核查每张保修单对应验收确认单
4. **保修金信息未上报**：竣工结算后未在15日内报送保修金信息 → 核查上报时间和内容完整性
5. **保修金预留不足**：分包合同未同比例预留保证金 → 核查分包结算中质量保证金扣留比例
6. **缺陷责任期超期**：未及时办理终止手续导致保修金无法回收 → 核查缺陷责任期起止时间和终止手续
7. **保修费用超额**：预留超过质保金10%上限 → 核查预留申请及审批记录
8. **常驻人员未到位**：工程交付1年内无保修人员 → 核查保修人员名单和驻场记录

### 关键指标

| 指标 | 计算方式 | 达标标准 |
|------|---------|---------|
| 保修响应及时率 | 48小时内响应维修次数/总保修次数×100% | 100% |
| 保修销项率 | 已销项保修单数/总保修单数×100% | ≥100%（1年内） |
| 客户满意度 | 满意评价次数/回访总次数×100% | ≥90% |
| 保修金回收率 | 已回收保修金/应收保修金×100% | 100% |
| 保修费用预留合规率 | 合规预留次数/总预留次数×100% | 100%（上限10%） |
| 信息上报及时率 | 15个工作日内上报次数/应上报次数×100% | 100% |
| 缺陷责任期终止办理率 | 已办终止数/应办终止数×100% | 100% |
