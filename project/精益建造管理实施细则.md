## 核心条款（要点充沛，引用原文关键表述）

### 第X条（目的）

> "紧跟中建集团'一创五强'战略目标，践行'三减两降一提升'建造理念，通过全过程精细管理，持续提升工程建造品质、缩短建造周期，降低建造成本，提高综合收益，打造客户满意工程，以完美履约助力市场开拓。"

### 第X条（术语定义）

| 术语 | 定义 |
|------|------|
| 精益建造 | 综合生产管理理论、建筑管理理论以及建筑生产的特殊性，面向建筑产品的全生命周期，持续地减少和消除浪费，最大限度地满足顾客的要求的系统性的方法 |
| 七大体系 | 以科技建造体系为引领与基础，以本质安全建造体系、智慧建造体系、绿色建造体系为保障，实施以客户敏感点为核心的快速建造体系、精益成本建造体系和精品建造体系 |
| 三减两降一提升 | 减少多余工序、减少工作面闲置、减少资源浪费、降低质量风险、降低造价成本、提升工程品质 |

### 第X条（管理原则）

| 原则 | 说明 |
|------|------|
| 顾客价值最大化原则 | 以顾客的最大化价值为目标，用最小的资源投入（人力/设备/资金/材料/时间/空间）满足项目预定目标 |
| 浪费最小化原则 | 减少多余工序/工作面闲置/资源浪费，提高一次成优率，减少一次性措施投入，追求"零浪费/零库存/零缺陷/零事故/零返工/零窝工" |
| 分类实施原则 | 结合项目实施精益建造的条件和背景存在差异，建立精益建造项目分类标准，实施分类管理，并按分类要求实施差异化考核 |

## 四维宣贯表

### 做什么（要点充沛，8-15条）

1. **七大体系建设**：科技建造体系（引领基础）、本质安全建造体系、智慧建造体系、绿色建造体系、快速建造体系、精益成本建造体系、精品建造体系。
2. **精益建造管理标准制定**：公司总部按局发布指南制定精益建造标准；分区域/分产品线制定底线标准和引领标准；每半年完成一次标准的总结与提升。
3. **项目分类管理**：根据实施条件和背景将项目分为A/B/C三类（A全面实施/B创造条件全面实施/C部分实施）。
4. **项目精益建造实施计划制定**：项目在开展"实施策划"/"施工组织设计"/"首次资源配置"的同时制定《项目精益建造措施计划表》。
5. **计划评审与备案**：《项目精益建造措施计划表》随同"项目实施策划书"一并提交分公司/事业部评审，报公司项目管理部备案。
6. **措施效果分析**：项目部应在每个子分部精益建造措施实施完毕后15天内完成实施效果分析并编制分析报告。
7. **分公司年度计划**：分公司制定《分公司年度精益建造实施计划》，建立精益建造项目台账，明确示范项目清单、关键节点目标。
8. **分类动态管理**：A类项目须100%纳入示范项目清单；B类项目每季度评估实施条件；C类项目核心措施应用尽用。
9. **月度巡检季度考核**：分公司实施"月度巡检/季度考核"机制，重点核查工序穿插/资源投入/成本偏差等。
10. **年度总结提炼**：分公司每年组织项目完成《精益建造实施成果总结》，提炼快速建造措施/技术优化项/成本节约点/管理改进建议。
11. **政策对接**：分公司主动对接政府主管部门争取结构分段验收/使用爬架等政策支持，建立区域绿色通道。
12. **培训实施**：公司制定精益建造年度培训计划并实施。
13. **考核评价**：将精益建造实施情况纳入履约品质检查和分公司管理行为检查，纳入年底分公司年度综合管理评价。
14. **体系迭代升级**：总结提炼精益建造优秀经验，推动精益建造体系迭代升级。

### 怎么做（操作流程详细展开，10-20步）

#### 一、精益建造管理体系建设流程

**步骤1：标准制定**
- 发起部门：项目管理部/科技质量部/商务管理部/安全生产监督管理部/企业策划部/信息部
- 操作动作：
  - 精益建造工作小组建立健全精益建造管理体系
  - 制定精益建造标准，明确不同产品线底线标准和引领标准
  - 重点提升分部分项工程精益建造关键点标准
  - 每半年完成一次标准的总结与提升
- 输出文件：精益建造标准文件
- 签字审批：精益建造工作小组

**步骤2：培训计划制定与实施**
- 发起部门：精益建造工作小组
- 操作动作：制定精益建造年度培训计划并组织实施
- 输出文件：年度培训计划/培训记录

#### 二、项目分类管理流程

**步骤3：项目分类评估**
- 发起部门：分公司/事业部
- 操作动作：根据项目实施条件和背景，将项目分为A/B/C三类
- 分类标准：
  - A类（全面实施）：配合各方对精益建造实施条件全部认可
  - B类（创造条件全面实施）：配合各方部分认可精益建造实施条件
  - C类（部分实施）：配合各方不认可精益建造实施条件
- 输出文件：项目分类清单

**步骤4：分类实施要求落实**
- 发起部门：分公司/事业部/项目部
- 操作动作：
  - A类项目：全面落实精益建造管理标准内容
  - B类项目：创造条件，争取全面落实
  - C类项目：引用精益建造理念，局部落实精益建造管理标准内容

#### 三、项目精益建造实施计划流程

**步骤5：实施计划制定**
- 发起部门：项目部
- 操作动作：项目在开展"实施策划"/"施工组织设计"/"首次资源配置"工作的同时，制定完成《项目精益建造措施计划表》
- 结合中标价格或预算价格对每项措施进行评估，确定精益建造措施后预估可获取的预期收益
- 输出文件：《项目精益建造措施计划表》
- 签字审批：项目经理

**步骤6：计划评审**
- 发起部门：分公司/事业部
- 操作动作：对《项目精益建造措施计划表》进行评审
- 输出文件：评审意见

**步骤7：备案**
- 发起部门：分公司/事业部
- 操作动作：将评审后的《项目精益建造措施计划表》报公司项目管理部备案
- 输出文件：备案回执

**步骤8：措施落实**
- 发起部门：项目部
- 操作动作：《项目精益建造措施计划表》中确定的各项措施应落实到施工组织设计和施工方案中
- 指定各项措施的具体责任人
- 输出文件：落实记录

**步骤9：效果分析**
- 发起部门：项目部
- 操作动作：每个子分部精益建造措施实施完毕后15天内完成实施效果分析
- 编制分析报告
- 输出文件：效果分析报告

#### 四、分公司管理流程

**步骤10：年度计划制定**
- 发起部门：分公司/事业部
- 操作动作：制定《分公司年度精益建造实施计划》
- 建立精益建造项目台账
- 明确示范项目清单/关键节点目标
- 输出文件：《分公司年度精益建造实施计划》

**步骤11：分类动态管理**
- 发起部门：分公司/事业部
- 操作动作：
  - A类项目须100%纳入示范项目清单
  - B类项目每季度评估实施条件
  - C类项目要求对核心措施应用尽用
- 输出文件：分类动态管理记录

**步骤12：月度巡检季度考核**
- 发起部门：分公司/事业部
- 操作动作：实施"月度巡检/季度考核"机制
- 结合《项目精益建造措施计划表》跟踪措施落地情况
- 重点核查工序穿插/资源投入/成本偏差等
- 输出文件：巡检记录/季度考核报告

**步骤13：年度总结**
- 发起部门：分公司/事业部
- 操作动作：每年组织项目完成《精益建造实施成果总结》
- 内容：快速建造措施/技术优化项/成本节约点/管理改进建议
- 报公司项目管理部
- 输出文件：《精益建造实施成果总结》

**步骤14：政策对接**
- 发起部门：分公司/事业部
- 操作动作：主动对接政府主管部门
- 争取结构分段验收/使用爬架等政策支持
- 建立区域绿色通道
- 输出文件：政策对接记录

#### 五、考核评价流程

**步骤15：公司考核**
- 发起部门：公司项目管理部
- 操作动作：结合履约品质检查和分公司管理行为检查，对分公司精益建造实施情况进行考核评价
- 纳入年底分公司年度综合管理评价
- 输出文件：考核评价报告

### 常见问题 / 风险点

1. **项目分类不准确**：未根据实际实施条件对项目进行准确分类，导致实施要求不匹配
2. **实施计划空洞**：计划内容缺乏针对性，与项目实际情况脱节
3. **措施未落实到施工方案**：计划中的精益建造措施未真正融入施工组织设计和施工方案
4. **效果分析不及时**：子分部完成后超过15天未完成效果分析报告
5. **月度巡检流于形式**：巡检发现问题但未跟踪整改
6. **分公司年度计划与实际脱节**：年度计划目标过高或过低
7. **示范项目示范作用不足**：A类项目未真正起到示范引领作用
8. **体系更新滞后**：标准未按每半年进行一次总结与提升

### 关键指标

| 指标名称 | 计算方式 | 达标标准 |
|------|------|------|
| 项目分类准确率 | 分类准确的项目数/参与分类项目总数×100% | 100% |
| A类项目示范率 | A类项目数/应纳入示范项目总数×100% | 100% |
| 实施计划制定及时率 | 按时制定《项目精益建造措施计划表》的项目数/应制定项目总数×100% | 100% |
| 计划评审备案率 | 完成评审并备案的项目数/应评审备案项目总数×100% | 100% |
| 效果分析按时完成率 | 按期完成效果分析报告的次数/应完成总次数×100% | 100% |
| 月度巡检开展率 | 按时开展月度巡检的次数/应开展总次数×100% | 100% |
| 季度考核开展率 | 按时开展季度考核的次数/应开展总次数×100% | 100% |
| 年度成果总结完成率 | 完成《精益建造实施成果总结》的分公司数/应完成总数×100% | 100% |
| 精益建造标准更新及时率 | 按时更新标准的次数/应更新总次数×100% | 100%（每半年） |
| 成本节约率 | 精益建造措施实现成本节约金额/计划成本×100% | 越高越好 |
| 工期优化率 | 精益建造措施实现工期优化天数/合同工期×100% | ≥10%（内控工期较合同工期优化10%） |
